## 基本用法

Draggable 支持两种手势模式：`immediate`（触摸后立即开始拖拽）和 `longpress`（长按后开始拖拽）。
你也可以控制拖拽结束后的位置是重置，还是保留当前的 transform 位置。

export const draggableBasicMeta = {
  highlight: '16-17',
  description: 'highlight resetOnEnd and trigger',
  content: ['resetOnEnd={true}', "trigger='immediate'"],
};

## 带边界限制

可拖拽区域可以通过 `min(max)TranslateX/Y` 属性进行限制。

export const draggableWithBoundsMeta = {
  highlight: '18-21',
  description: 'highlight how the bounds are specified',
  content: [
    'minTranslateX={-100}',
    'minTranslateY={-100}',
    'maxTranslateX={100}',
    'maxTranslateY={100}',
  ],
};

## 结构

整个组件都可以被拖拽。

```tsx
<Draggable />
```

只有 `DraggableArea` 会响应拖拽手势。

```tsx
<DraggableRoot>
  <DraggableArea />
</DraggableRoot>
```
