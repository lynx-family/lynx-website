## Keyboard Avoidance

Prevent the on-screen keyboard from obscuring the input field by automatically adjusting the view when the keyboard appears

## Structure

```jsx
<KeyboardAwareRoot>
  <view className='container'>
    <KeyboardAwareResponder>
      <KeyboardAwareTrigger>
          <Input />
      </KeyboardAwareTrigger>
      <KeyboardAwareTrigger>
          <TextArea />
      </KeyboardAwareTrigger>
    </KeyboardAwareResponder>
  </view>
<KeyboardAwareRoot>
```
