# `<FeedList>`

{intros['lynx-ui-feed-list']}
import { PageTabs, PageTab } from '@theme';

import Introduction from '../../IntroDocs/lynx-ui-feed-list/Introduction.mdx';
import APIReference from './APIReference.mdx';

A scrollable container based on [`<List>`](/lynx-ui/Components/List.html).
Adds refresh and bounceable capabilities implemented by MTS, and also supports custom footers for LoadMore status and data load completion status.

:::info

You can check the [source code](https://github.com/lynx-family/lynx-ui/tree/main/packages/lynx-ui-feed-list) to understand how it works, and you're welcome to submit MRs to enhance its capabilities;

Moreover, you can also implement your own tabs functionality with [MTS](https://lynx.bytedance.net/api/lynx-api/main-thread.html).

:::
