A radioGroup is a set of checkable buttons—known as radio buttons—where only one can be selected at a time.

## Basic Usage

<Go
  img=""
  example="RadioGroup"
  defaultEntryName="RadioGroupBasic"
  defaultFile="Basic/index.tsx"
  entry="Basic"
/>

## Structure

```jsx
<RadioGroupRoot>
  <Radio>
    <RadioIndicator />
  </Radio>
</RadioGroupRoot>
```
