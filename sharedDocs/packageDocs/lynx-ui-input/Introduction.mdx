## Structure

```jsx
<KeyboardAwareRoot>
  <view className='container'>
    <KeyboardAwareResponder>
      <KeyboardAwareTrigger>
          <Input
            style={{
              width: '100%',
              padding: '10px',
              backgroundColor: 'yellow',
            }}
          />
      </KeyboardAwareTrigger>
      <KeyboardAwareTrigger>
          <TextArea
            style={{
              width: '100%',
            }}
            placeholder="textareaId:'textarea2'"
          />
      </KeyboardAwareTrigger>
    </KeyboardAwareResponder>
  </view>
<KeyboardAwareRoot>
```
