## 结构

```jsx
<PopoverRoot>
  <PopoverTrigger />
  <PopoverAnchor>
    <PopoverPositioner>
      <PopoverContent />
      <PopoverArrow />
    </PopoverPositioner>
  </PopoverAnchor>
</PopoverRoot>
```

## 样式

本节展示组件中与动画相关的生命周期。当 `transition` 属性为 `true` 时，额外的 className 会被添加到组件的 className 上，而 render props 始终可用。
