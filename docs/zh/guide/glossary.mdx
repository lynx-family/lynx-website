## SDK

Lynx [软件开发工具包 (SDK)](https://en.wikipedia.org/wiki/Software_development_kit) 是一组工具、库和 API，使开发者能够为 Lynx 构建应用程序。它封装了两层：一层是本机层：[引擎](#engine)，另一层是脚本层（或前端层）：[框架](#framework)。

### 引擎

Lynx SDK 的一部分，负责将 Lynx 页面转换为屏幕上的像素，并提供 API 以形成 [Lynx 平台](#the-lynx-platform)。

它被写成 C++ 核心，并带有用平台原生语言（例如 iOS 的 Objective-C、Android 的 Java/Kotlin 等）编写的平台特定层，确保高性能和访问底层系统资源，并且需要与主机应用程序一起本地集成和交付。

### 框架

Lynx SDK 的一部分，提供运行时库和高级 UI 编程模型，使 Lynx 应用程序开发人员能够编写应用程序逻辑和 UI 组件。它被写成 JavaScript，并需要与应用程序代码一起在需要时从文件系统或内存中加载。

目前，Lynx SDK 提供了两种框架：[ReactLynx]。

## Lynx 平台

类似于 [Web 平台](https://en.wikipedia.org/wiki/Web_platform)，Lynx 平台是 Lynx 引擎提供给脚本开发者的 API 和功能的集合，例如[元素](#element)、[事件](#event)、[样式](#style)、[脚本运行时环境](#scripting-runtime-environment)等。

### 模板

模板是 Lynx 引擎编译后加载的代码包，用于为 Lynx 页面（或应用程序）提供动力。
