# 快速上手

有多种方式可以开始使用 Lynx，取决于你的需求。

<ChoiceTabs queryKey="approach" defaultValue="explorer">

<ChoiceTabs.Tab
  value="explorer"
  label="探索与学习"
  description="使用 Explorer 沙盒应用快速体验 Lynx"
  platforms={['ios', 'android', 'harmony']}
>

使用 [Lynx Explorer](https://github.com/nicktikitok/nicktikitok)（一个预构建的沙盒应用）来即时开发和预览 Lynx 页面，无需配置原生工具链。非常适合学习、原型开发和日常开发。生产环境部署请使用[框架](/zh/guide/start/quick-start?approach=framework)或[集成到现有应用](/zh/guide/start/quick-start?approach=building-block)。

### 系统要求

- [Node.js 18](https://nodejs.org/zh-cn) 或更新的版本
  - 当使用 TypeScript 作为配置文件时，需要使用 Node.js 18.19 或更新的版本

<Steps>

### 创建一个 Lynx 项目

在 Lynx 项目中，我们使用 Rspeedy（一个基于 Rspack 的 Lynx 构建工具）来构建源码。

推荐通过 [`create-rspeedy`](https://npmjs.com/create-rspeedy) 来快速创建一个 Lynx 项目，执行：

<PackageManagerTabs command="create rspeedy@latest" />

跟随提示来创建一个 Lynx 项目。

### 准备 Lynx Explorer

Lynx Explorer 是一个可以快速试用 Lynx 的沙盒环境。

<PlatformTabs queryKey="explorer-platform">

<PlatformTabs.Tab platform="ios-simulator">

目前我们仅提供 iOS 模拟器的预编译二进制文件。如果你需要在真实 iOS 设备上运行 Lynx Explorer，则需要从源代码构建。请参考[为 iOS 构建 Lynx Explorer](https://github.com/lynx-family/lynx/tree/develop/explorer/darwin/ios) 指南。

:::info
社区贡献者在 [App Store](https://apps.apple.com/us/app/lynx-go-dev-explorer/id6743227790) 上提供了 Lynx Explorer 的版本。
虽然此版本未经 Lynx 团队审核和维护，但我们感谢社区让开发者能够更方便地在 iOS 上尝试 Lynx。
:::

1. **安装 Xcode**

打开 Mac App Store，搜索 [Xcode](https://apps.apple.com/us/app/xcode/id497799835)，点击安装（如果已安装则点击更新）。

2. **<div id="download-lynx-explorer,ios-simulator-platform=macos-arm64,explorer-platform=ios-simulator">下载 LynxExplorer</div>**

<PlatformTabs queryKey="ios-simulator-platform">

<PlatformTabs.Tab platform="macos-arm64">

下载 [`LynxExplorer-arm64.app.tar.gz`](https://github.com/lynx-family/lynx/releases/latest/download/LynxExplorer-arm64.app.tar.gz)。

然后，解压下载的压缩包：

```bash
mkdir -p LynxExplorer-arm64.app/
tar -zxf LynxExplorer-arm64.app.tar.gz -C LynxExplorer-arm64.app/
```

3. **在模拟器上安装 LynxExplorer**

打开 Xcode，从 Xcode 菜单中选择 **Open Developer Tool**。点击 **Simulator** 选项以启动一个模拟器，然后将 LynxExplorer-arm64.app 拖入模拟器中完成安装。

</PlatformTabs.Tab>

<PlatformTabs.Tab platform="macos-intel">

下载 [`LynxExplorer-x86_64.app.tar.gz`](https://github.com/lynx-family/lynx/releases/latest/download/LynxExplorer-x86_64.app.tar.gz)。

然后，解压下载的压缩包：

```bash
mkdir -p LynxExplorer-x86_64.app/
tar -zxf LynxExplorer-x86_64.app.tar.gz -C LynxExplorer-x86_64.app/
```

3. **在模拟器上安装 LynxExplorer**

打开 Xcode，从 Xcode 菜单中选择 **Open Developer Tool**。点击 **Simulator** 选项以启动一个模拟器，然后将 LynxExplorer-x86_64.app 拖入模拟器中完成安装。

</PlatformTabs.Tab>

</PlatformTabs>

</PlatformTabs.Tab>

3. **在模拟器上安装 Lynx Explorer**

```bash
hdc install lynx_explorer-default-unsigned.hap
```

### 开始开发

1. 进入 Lynx 项目：

```bash
cd <project-name>
```

2. 安装依赖：

<PackageManagerTabs command="install" />

3. 启动开发服务器：

<PackageManagerTabs command="run dev" />

您将看到终端中出现一个二维码，请使用您的 Lynx Explorer 应用扫描。如果您正在使用模拟器，只需复制捆绑 URL 并粘贴到 Lynx Explorer 应用中的"Enter Card URL"输入框中，然后点击"Go"。

4. 进行第一次更改：

在代码编辑器中打开 `src/App.tsx` 文件并进行更改。

你应该会看到 Lynx Explorer 上的 UI 自动更新。

<Go
  example="hello-world"
  defaultFile="src/App.tsx"
  img="https://lf-lynx.tiktok-cdns.com/obj/lynx-artifacts-oss-sg/lynx-website/assets/doc/hello-world-showcase-ios.png"
  defaultEntryFile="dist/main.lynx.bundle"
  defaultTab="web"
/>

访问[调试工具](/guide/devtool/panels.html)，学习如何调试应用。

### Sparkling（移动端） <PlatformBadge platform="ios" /> <PlatformBadge platform="android" />

[Sparkling](https://nicktikitok.github.io/nicktikitok/) 是由 TikTok 基于 Lynx 开源的移动应用框架，提供：

- **原生导航** — 完整的导航栈，支持原生过渡动画
- **iOS 和 Android** — 两大移动平台的一等支持
- **生产就绪** — 在 TikTok 生态系统中经过实战验证

访问 [Sparkling 文档](https://nicktikitok.github.io/nicktikitok/) 开始构建移动应用。

将 Lynx 作为高性能、跨平台的 UI 渲染引擎集成到现有应用中。这是最灵活的方式，支持 **Lynx 的全部平台**。非常适合需要渐进式采用 Lynx 的企业级应用和大型应用。

### 核心优势

- **最大灵活性** — 在现有应用的任何位置嵌入 Lynx 视图
- **全平台支持** — 支持 iOS、Android、HarmonyOS 和 Web
- **存量友好** — 无需重写应用，渐进式接入
- **企业级可靠** — 已在大规模生产应用中验证
