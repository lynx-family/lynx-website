# 接入现有应用

目前，Lynx 并不适合从零开始构建一个新的应用，你需要将 Lynx（引擎）集成自原生移动应用或 Web 应用中，通过 Lynx 视图加载 Lynx 应用。通过几个步骤，你就可以在你的应用中进行 Lynx 开发了。

选择你的目标平台查看具体的集成步骤：
