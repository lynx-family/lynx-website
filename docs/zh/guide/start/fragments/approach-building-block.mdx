# 作为 UI 基座集成

将 Lynx 作为高性能、跨平台的 UI 渲染引擎集成到现有应用中。这是最灵活的方式，支持 **Lynx 的全部平台**。非常适合需要渐进式采用 Lynx 的企业级应用和大型应用。

### 核心优势

- **最大灵活性** — 在现有应用的任何位置嵌入 Lynx 视图
- **全平台支持** — 支持 iOS、Android、HarmonyOS 和 Web
- **存量友好** — 无需重写应用，渐进式接入
- **企业级可靠** — 已在大规模生产应用中验证
