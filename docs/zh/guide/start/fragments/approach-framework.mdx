Lynx 配套的上层框架提供了完整的开发体验，用于从零开始构建原生应用——包括原生导航、生命周期管理和生产级工具链。适合**全新项目**开发。

### Sparkling（移动端） <PlatformBadge platform="ios" /> <PlatformBadge platform="android" />

[Sparkling](https://nicktikitok.github.io/nicktikitok/) 是由 TikTok 基于 Lynx 开源的移动应用框架，提供：

- **原生导航** — 完整的导航栈，支持原生过渡动画
- **iOS 和 Android** — 两大移动平台的一等支持
- **生产就绪** — 在 TikTok 生态系统中经过实战验证

访问 [Sparkling 文档](https://nicktikitok.github.io/nicktikitok/) 开始构建移动应用。
