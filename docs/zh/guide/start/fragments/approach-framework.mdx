基于 Lynx 构建的 **应用框架（App Framework）** 提供端到端的开发体验，帮助你从零交付一个完整的原生应用，内置原生导航、生命周期管理和生产级工具链，适合**全新**项目。

**[Sparkling](https://tiktok.github.io/sparkling/)** 是 TikTok 基于 Lynx 开源的**移动端**框架，将驱动 TikTok 的跨平台基础设施带给开发者。它提供类型安全的 JavaScript 到原生桥接、基于 scheme 的多页面路由以及项目脚手架，从同一份代码包同时部署到 **iOS 和 Android**，并已在 TikTok 生态中经过大规模生产验证。访问 [Sparkling 文档](https://tiktok.github.io/sparkling/) 开始构建移动应用。

**Lynxtron** 是即将推出的**桌面端**框架，一个轻量的 Electron 替代方案，以 Lynx 渲染同一份 UI 跨越 **macOS、Windows、Linux 和 Web**，并支持通过 Node-API 扩展原生模块。即将开源，敬请期待。
