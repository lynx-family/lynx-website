# @lynx-js/lynxtron-rebuild

`@lynx-js/lynxtron-rebuild` 用于将 npm Node.js 原生模块重新编译到 Lynxtron runtime ABI。它对应 Electron 生态中的 `@electron/rebuild`，但目标 runtime 是 Lynxtron。

## 安装

```bash
npm install -D @lynx-js/lynxtron-rebuild
```

## 用法

```bash
npx @lynx-js/lynxtron-rebuild
```

通常在安装或更新原生模块后执行，并在打包前确认重建产物可被 Lynxtron runtime 加载。

## 适用场景

| 场景                               | 是否需要                                                        |
| ---------------------------------- | --------------------------------------------------------------- |
| 纯 JavaScript npm 包               | 不需要 rebuild                                                  |
| 包含 `.node` 二进制的 npm 原生模块 | 需要使用 `@lynx-js/lynxtron-rebuild`                            |
| 从 Electron 项目迁移过来的原生模块 | 需要从 Electron ABI 重建到 Lynxtron runtime ABI                 |
| Lynx 原生能力库                    | 通常通过 `npm run build:shared` 构建，不依赖 `lynxtron-rebuild` |

更多说明见 [Node.js 原生模块](/lynxtron/Native-Libraries/NodeJS-Native-Modules)。
