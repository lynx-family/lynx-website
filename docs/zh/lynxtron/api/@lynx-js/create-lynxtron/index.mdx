# @lynx-js/create-lynxtron

`create-lynxtron` 是 Lynxtron 官方应用模板的项目初始化器。它会复制模板、写入项目名、替换模板中的 workspace 依赖版本，并根据选项保留或移除 Web 支持相关代码。

发布包名为 `@lynx-js/create-lynxtron`，推荐通过 `npm create @lynx-js/lynxtron@latest` 创建项目。

## 用法

```bash
npm create @lynx-js/lynxtron@latest my-app
npm create @lynx-js/lynxtron@latest my-app -- --web
npm create @lynx-js/lynxtron@latest my-app -- --no-web
```

## 参数

| 参数             | 说明                                                                      |
| ---------------- | ------------------------------------------------------------------------- |
| `[project-name]` | 要创建的项目名或目标路径。未传入时会交互式询问，默认值为 `lynxtron-app`。 |
| `--web`          | 默认包含 Web 支持，也就是生成 Symmetric Host 相关代码。                   |
| `--no-web`       | 不生成 Web 支持相关代码。                                                 |

## 行为

- 如果没有传入项目名或 Web 支持选项，CLI 会进入交互式询问。
- 如果目标目录已存在且不为空，CLI 会询问是否覆盖。
- 默认模板是 Lynxtron 桌面应用模板，包含 Node.js 主进程和 Lynx UI 代码。
- 生成完成后，通常执行 `npm install` 和 `npm run dev` 开始开发。
