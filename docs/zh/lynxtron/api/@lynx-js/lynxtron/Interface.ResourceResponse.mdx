# 接口：ResourceResponse

`ProtocolHandler` 返回的响应对象。

运行时只会接受同时包含 `statusCode` 和 `Buffer` 类型 `data` 的响应。无效响应会回退到正常的
Lynx 资源加载流程。

## 属性

### url

```ts
url: string;
```

响应 URL。

#### 定义于

lynxtron/apis/api/protocol.d.ts

---

### statusCode

```ts
statusCode: number;
```

响应的 HTTP 风格状态码。

#### 定义于

lynxtron/apis/api/protocol.d.ts

---

### data

```ts
data: Buffer;
```

Node.js `Buffer` 形式的响应体。

#### 定义于

lynxtron/apis/api/protocol.d.ts
