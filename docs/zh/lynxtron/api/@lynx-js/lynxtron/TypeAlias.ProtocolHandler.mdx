# 类型别名：ProtocolHandler

```ts
type ProtocolHandler = (
  request: ResourceRequest,
) => ResourceResponse | Promise<ResourceResponse> | false | undefined;
```

用于处理已注册 URL scheme 下的 Lynx 资源请求。

返回 `ResourceResponse` 或 `Promise<ResourceResponse>` 时表示提供该资源。返回 `false`
或 `undefined` 时，会回退到正常的 Lynx 资源加载流程。响应必须包含 `statusCode` 和
`Buffer` 类型的 `data`；无效响应会回退到正常加载流程。

#### 参数

| 参数      | 类型              |
| --------- | ----------------- |
| `request` | `ResourceRequest` |

#### 返回

`ResourceResponse | Promise<ResourceResponse> | false | undefined`
