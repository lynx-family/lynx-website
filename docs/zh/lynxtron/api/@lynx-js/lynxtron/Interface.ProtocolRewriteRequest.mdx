# 接口：ProtocolRewriteRequest

传给 `ProtocolRequestRewriter` 的请求对象。

请求重写会先于 `protocol.handle()` 注册的处理函数和 Lynx 默认资源加载流程执行。

## 属性

### scheme

```ts
scheme: string;
```

请求中已规范化的 URL scheme。

#### 定义于

lynxtron/apis/api/protocol.d.ts

### url

```ts
url: string;
```

原始资源 URL。
