# 类型别名：ProtocolRequestRewriter

```ts
type ProtocolRequestRewriter = (
  request: ProtocolRewriteRequest,
) => string | false | undefined;
```

用于在 `protocol.handle()` 注册的处理函数和 Lynx 默认资源加载流程运行前重写资源请求 URL。

> [!NOTE]
> 这是 Lynxtron 特有 API。

返回字符串时，该字符串会作为新的 URL 使用。返回 `false`、`undefined` 或任意非字符串值时，
则保留原始 URL。

#### 参数

| 参数      | 类型                     |
| --------- | ------------------------ |
| `request` | `ProtocolRewriteRequest` |

#### 返回

`string | false | undefined`

#### 定义于

lynxtron/apis/api/protocol.d.ts
