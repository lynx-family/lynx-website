# @lynx-js/lynxtron/web-host

Web host 类型入口，用于在 Web host 中运行 Lynx 时配置兼容 Lynxtron 的 `NativeModules`。

## 函数

| 函数                                                                                           | 描述                           |
| ---------------------------------------------------------------------------------------------- | ------------------------------ |
| [setupSymmetricHost](/lynxtron/api/@lynx-js/lynxtron/web-host/Function.setupSymmetricHost.mdx) | 设置 Web 侧的 Symmetric Host。 |

## 接口

| 接口                                                                                                      | 描述                        |
| --------------------------------------------------------------------------------------------------------- | --------------------------- |
| [BackgroundServiceConfig](/lynxtron/api/@lynx-js/lynxtron/web-host/Interface.BackgroundServiceConfig.mdx) | `nodejs` 后台服务配置。     |
| [HostConfig](/lynxtron/api/@lynx-js/lynxtron/web-host/Interface.HostConfig.mdx)                           | bridge 和 nodejs 模块配置。 |
| [MainThreadBridge](/lynxtron/api/@lynx-js/lynxtron/web-host/Interface.MainThreadBridge.mdx)               | 主线程 bridge 方法映射。    |
