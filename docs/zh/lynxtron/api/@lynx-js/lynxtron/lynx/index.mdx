# @lynx-js/lynxtron/lynx

Lynxtron 在 Lynx BTS 和 Lynx UI 代码中使用的 Lynx 侧 TypeScript 类型入口。

## 接口

| 接口                                                                                              | 描述                                                                                                                            |
| ------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| [LynxtronNodejs](/lynxtron/api/@lynx-js/lynxtron/lynx/Interface.LynxtronNodejs.mdx)               | Lynx BTS 中 [`NativeModules.nodejs`](/lynxtron/api/@lynx-js/lynxtron/lynx/Interface.LynxtronNodejs.mdx) 的类型。                |
| [LynxtronNodejsExposed](/lynxtron/api/@lynx-js/lynxtron/lynx/Interface.LynxtronNodejsExposed.mdx) | 应用可扩展的 [`NativeModules.nodejs.exposed`](/lynxtron/api/@lynx-js/lynxtron/lynx/Interface.LynxtronNodejs.mdx#exposed) 类型。 |
