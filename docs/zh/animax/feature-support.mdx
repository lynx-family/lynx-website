# 功能支持

这个文档列出了 AnimaX 对常用 After Effects 特性的支持情况，并对照 Lottie Android 与 Lottie iOS 的能力差异。

表格中的 AnimaX 版本表示对应特性开始获得 AnimaX 支持的版本号。例如，`1.0+` 表示该特性从 AnimaX 1.0 起支持。更多版本发布信息可以参考 [AnimaX releases](https://github.com/lynx-family/animax/releases)。

Lottie 的功能支持数据参考 [LottieFiles Supported Features](https://lottiefiles.com/supported-features)。
