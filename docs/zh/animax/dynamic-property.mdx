# 动态属性

在 Lynx 应用中，AnimaX 通过 JS NAPI 接口向前端开发者提供运行时更新动画属性的功能。

:::info

本文中的 Lynx 组件示例使用 [`<animax-view>`](/api/elements/built-in/animax-view.html) 集成能力。原生应用可以直接使用 Android 与 iOS API 接入。

动画属性分为 图层属性 和 资源属性。其中 图层属性 包括 transform、可见性等，可以在动画播放期间动态修改。而 资源属性 包括图片、文字资源和视频资源等，只能在动画播放之前修改。

:::

## 通过前端组件使用动态属性

我们定义了动态属性的相关类型，并封装了动态属性相关的接口。

在实践中，推荐将动态属性调用封装在带类型的前端组件中，让资源更新、图层更新与播放生命周期处理集中在一处。

### 使用通配符更新图层属性

AnimaX 支持通过通配符更新图层属性。当 `layerName` 传入 `**` 时，AnimaX 会尝试更新所有图层的对应属性；当传入 `*` 时，则匹配单个层级下的所有图层。
如果 `layerName` 命中多个同名图层，所有匹配图层都会被选中并更新。
