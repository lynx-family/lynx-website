# AnimaXLoaderCompletionHandler

`AnimaXLoaderCompletionHandler` 是资源加载完成回调。加载器完成后调用它，将 [AnimaXLoaderResponse](./animax-loader-response) 传回 AnimaX。

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

## Block 签名

```objc
typedef void (^AnimaXLoaderCompletionHandler)(AnimaXLoaderResponse *response);
```

## 参数

| 名称       | 类型                                                   | 说明       |
| ---------- | ------------------------------------------------------ | ---------- |
| `response` | [`AnimaXLoaderResponse`](./animax-loader-response) `*` | 加载响应。 |

</PlatformTabs.Tab>
<PlatformTabs.Tab platform="android">

## 完成回调

### onComplete
