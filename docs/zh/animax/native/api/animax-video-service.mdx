# AnimaXVideoService

`AnimaXVideoService` 对应 Android 的 `IAnimaXVideoService`，用于在 Android 上接入自定义视频播放器。AnimaX 在需要创建自定义播放器时会通过此服务获取 `IVideoPlayer` 实例。

## 创建播放器

### createVideoPlayer

创建自定义视频播放器实例。返回 `null` 时回退到默认播放器。

**参数**

| 名称           | 类型                | 说明             |
| -------------- | ------------------- | ---------------- |
| `nativePlayer` | `long`              | 原生播放器句柄。 |
| `config`       | `VideoPlayerConfig` | 视频播放器配置。 |

**返回值**

`IVideoPlayer` 或 `null`。

## 注册方式

```java
AnimaXServiceCenter.inst().registerService(
    IAnimaXVideoService.class,
    myVideoService);
```
