# AnimaXComposition

`AnimaXComposition` 是已解析的动画合成对象，可通过 [AnimaXCompositionFactory](./animax-composition-factory) 从 JSON 或 URI 异步创建。创建完成后，可以将它传给 [AnimaXView](./animax-view) 的 `setComposition`，用于预加载和复用动画解析结果。

## 帧信息

### startFrame

返回动画合成的起始帧。

**返回值**

`CGFloat`

### endFrame

返回动画合成的结束帧。

**返回值**

`CGFloat`

<PlatformTabs.Tab platform="android">

## 帧信息

### getStartFrame

返回动画合成的起始帧。底层对象已释放时返回 `0`。

**返回值**

`long`

### getEndFrame

返回动画合成的结束帧。底层对象已释放时返回 `0`。

**返回值**

`long`

## 原生资源

### getNativePtr

返回当前合成对象持有的 native 指针。底层对象已释放时返回 `0`。

**返回值**

`long`

### release

立即释放当前合成对象持有的 native 资源。释放后不应继续复用当前 `AnimaXComposition`。
