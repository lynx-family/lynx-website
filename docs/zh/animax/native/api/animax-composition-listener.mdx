# AnimaXCompositionListener

`AnimaXCompositionListener` 是 Android 上 [AnimaXCompositionFactory](./animax-composition-factory) 的解析结果回调。解析成功时返回 [AnimaXComposition](./animax-composition)，解析失败时返回错误信息。

## 解析结果

### onCompositionReady

动画合成解析完成。

**参数**

| 名称          | 类型                                        | 说明               |
| ------------- | ------------------------------------------- | ------------------ |
| `composition` | [`AnimaXComposition`](./animax-composition) | 已解析的动画合成。 |

### onCompositionFailed

动画合成解析失败。

**参数**

| 名称           | 类型     | 说明       |
| -------------- | -------- | ---------- |
| `errorMessage` | `String` | 错误信息。 |
