# AnimaXView

`AnimaXView` 是原生播放视图组件，负责承载渲染 surface、加载动画资源，并提供播放控制、事件监听和动态属性更新能力。

## 初始化

### initWithContext

使用 `AnimaXContext` 创建播放视图。适合需要通过 context 注入自定义 ability、服务或运行时配置的场景。

**参数**

| 名称      | 类型              | 说明             |
| --------- | ----------------- | ---------------- |
| `context` | `AnimaXContext *` | 动画实例上下文。 |

**返回值**

`AnimaXView *`

### initWithPlayer

使用已创建的 `AnimaXPlayer` 创建播放视图。适合播放器与视图生命周期需要分开管理的场景。

**参数**

| 名称     | 类型             | 说明             |
| -------- | ---------------- | ---------------- |
| `player` | `AnimaXPlayer *` | 底层播放器实例。 |

**返回值**

`AnimaXView *`

## 属性

### enableNativeTapLayerEvent

控制原生触摸事件是否触发图层点击检测。启用后可通过 `onTapLayers:` 接收命中的图层名称。

**类型**

`BOOL`

### ignoreAttachStatus

控制视图 attach 状态是否参与 surface 初始化和可见性更新。通常只在宿主视图生命周期特殊时使用。

**类型**

`BOOL`

### player

只读属性，用于访问底层播放器协议对象。

**类型**

`id<AnimaXPlayerProtocol>`

## 资源加载

### setSrc

加载动画资源。可以直接传入 URL 字符串，也可以指定 bundle，用于查找动画依赖的图片等子资源。

**可用签名**

| 签名               | 说明                         |
| ------------------ | ---------------------------- |
| `setSrc:`          | 从 URL 字符串加载动画资源。  |
| `setSrc:inBundle:` | 从指定 bundle 加载动画资源。 |

**参数**

| 名称     | 类型         | 说明              |
| -------- | ------------ | ----------------- |
| `src`    | `NSString *` | 动画资源地址。    |
| `bundle` | `NSBundle *` | 资源所在 bundle。 |

### setJson

从 JSON 字符串加载动画。需要从特定 bundle 解析依赖资源时，使用带 bundle 的重载。

**可用签名**

| 签名                | 说明                                                     |
| ------------------- | -------------------------------------------------------- |
| `setJson:`          | 从 JSON 字符串加载动画。                                 |
| `setJson:inBundle:` | 从 JSON 字符串加载动画，并使用指定 bundle 查找依赖资源。 |

**参数**

| 名称     | 类型         | 说明                 |
| -------- | ------------ | -------------------- |
| `json`   | `NSString *` | Lottie JSON 字符串。 |
| `bundle` | `NSBundle *` | 资源所在 bundle。    |

### setComposition

直接设置已解析的 `AnimaXComposition`。适合复用预加载的动画合成。

**参数**

| 名称          | 类型                  | 说明           |
| ------------- | --------------------- | -------------- |
| `composition` | `AnimaXComposition *` | 动画合成对象。 |

### setImageFolder

设置外部图片资源目录。应在 `setJson:` 或 `setSrc:` 前调用。

**参数**

| 名称          | 类型         | 说明           |
| ------------- | ------------ | -------------- |
| `imageFolder` | `NSString *` | 图片资源目录。 |

### setDynamicResource

启用或关闭播放期间的动态资源更新能力。

**参数**

| 名称     | 类型   | 说明                         |
| -------- | ------ | ---------------------------- |
| `enable` | `BOOL` | `YES` 表示启用动态资源更新。 |

## 动态属性

### updateLayerProperty

**参数**

| 名称           | 类型                         | 说明                 |
| -------------- | ---------------------------- | -------------------- |
| `keyPath`      | `AnimaXKeyPath *`            | 目标图层或属性路径。 |

| `propertyType` | `ResourcePropertyType`       | 资源属性类型。 |
| `value`        | `AnimaXValueParam *`         | 新属性值。     |
| `callback`     | `id<AnimaXPropertyCallback>` | 可选完成回调。 |

### addLayerPropertyCallback

为图层属性添加动态值回调。回调会在渲染过程中提供属性值，适合跟随业务状态实时更新动画。

| `keyPath`       | `AnimaXKeyPath *`            | 目标图层或属性路径。 |
| `valueCallback` | `AnimaXValueCallback *`      | 动态值回调。         |
| `callback`      | `id<AnimaXPropertyCallback>` | 可选完成回调。       |

### getKeysForKeyPath

**参数**

| `keyPath`  | `AnimaXKeyPath *`               | 查询用 key path。    |

## 构造函数

### AnimaXView

创建播放视图。代码创建时传入 `Context`；XML 布局创建时传入 `Context` 和 `AttributeSet`，并读取 `AnimaXView` XML 属性。

**可用签名**

| 签名                                              | 说明                                    |
| ------------------------------------------------- | --------------------------------------- |
| `AnimaXView(Context context)`                     | 使用默认 `NativeAbility` 创建播放视图。 |
| `AnimaXView(Context context, AttributeSet attrs)` | 从 XML 布局创建播放视图。               |

**参数**

| 名称      | 类型           | 说明             |
| --------- | -------------- | ---------------- |
| `context` | `Context`      | Android 上下文。 |
| `attrs`   | `AttributeSet` | XML 属性集合。   |

**返回值**

`AnimaXView`

## XML 属性

### app:animax_animationUrl

设置动画资源 URL，对应 `setSrc(String src)`。

### app:animax_autoPlay

设置加载完成后是否自动播放，对应 `setAutoPlay(boolean autoPlay)`。默认值为 `true`。

### app:animax_loop

设置是否循环播放，对应 `setLoop(boolean enable)`。默认值为 `false`。

### app:animax_objectFit

设置内容适配方式，对应 `setObjectFit(ObjectFit objectFit)`。默认值为 `contain`。

## 视图选项

### getPlayer

返回当前视图内部持有的 `IAnimaXPlayer`。如果需要通过 player 接口统一管理资源加载、播放控制或事件监听，可以从这里取得实例。

**返回值**

`IAnimaXPlayer`

### setIgnoreAttachStatus

控制视图是否忽略 attach 状态。启用后，视图不会依赖 Android attach 生命周期来判断是否需要创建临时 surface，适用于宿主已经自行管理可见性或挂载状态的场景。

**参数**

| 名称     | 类型      | 说明                          |
| -------- | --------- | ----------------------------- |
| `ignore` | `boolean` | `true` 表示忽略 attach 状态。 |

### setEnableTapLayerEvent

控制是否启用原生图层点击检测。启用后，`onTouchEvent` 会把触点传给 AnimaX，并通过 `onTapLayers(AnimaXTapParam param)` 返回命中的图层名称。

**参数**

| 名称     | 类型      | 说明                          |
| -------- | --------- | ----------------------------- |
| `enable` | `boolean` | `true` 表示启用图层点击事件。 |
