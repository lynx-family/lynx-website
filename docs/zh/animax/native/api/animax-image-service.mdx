# AnimaXImageService

`AnimaXImageService` 用于把 AnimaX 的图片处理接入宿主应用已有的图片基础设施。

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

## 图片解码

### loadImageData

将原始图片数据解码为 `UIImage`。返回 `nil` 表示解码未处理，AnimaX 会继续使用默认解码链路。

**参数**

| 名称   | 类型       | 说明           |
| ------ | ---------- | -------------- |
| `data` | `NSData *` | 原始图片数据。 |

**返回值**

`UIImage *` 或 `nil`。

## 注册方式

```objc
[[AnimaXServiceCenter sharedInstance] registerService:@protocol(AnimaXImageDecoderService)
                                             withImpl:myImageDecoder];
```

## 图片加载

### loadImage

异步加载图片。返回 `true` 表示当前服务已接管该请求，AnimaX 不再继续使用默认图片加载链路。

**参数**

| 名称                | 类型                                                                  | 说明                         |
| ------------------- | --------------------------------------------------------------------- | ---------------------------- |
| `request`           | [`AnimaXLoaderRequest`](./animax-loader-request)                      | 图片加载请求。               |
| `completionHandler` | [`AnimaXLoaderCompletionHandler`](./animax-loader-completion-handler) | 图片加载完成后用于回传结果。 |

**返回值**

`boolean`。返回 `true` 表示当前服务已处理该请求；返回 `false` 时 AnimaX 继续使用默认加载链路。

## 注册方式

```java
AnimaXServiceCenter.inst().registerService(
    IAnimaXImageService.class,
    myImageService);
```
