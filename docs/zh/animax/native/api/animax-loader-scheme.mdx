# AnimaXLoaderScheme

`AnimaXLoaderScheme` 用于标识资源加载来源。自定义 loader 通过它声明自己处理的资源类型。

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

## 枚举值

| 枚举值                      | 数值 | 说明            |
| --------------------------- | ---- | --------------- |
| `AnimaXLoaderSchemeAsset`   | `0`  | 应用包资源。    |
| `AnimaXLoaderSchemeFile`    | `1`  | 本地文件。      |
| `AnimaXLoaderSchemeHttp`    | `2`  | HTTP 资源。     |
| `AnimaXLoaderSchemeDataURL` | `3`  | Data URL 资源。 |

</PlatformTabs.Tab>
<PlatformTabs.Tab platform="android">

## 枚举值

| 枚举值        | 说明                       |
| ------------- | -------------------------- |
| `ASSET`       | 应用包资源。               |
| `FILE`        | 本地文件。                 |
| `HTTP`        | HTTP 资源。                |
| `DATA_URL`    | Data URL 资源。            |
| `CONTENT_URI` | Android Content URI 资源。 |

</PlatformTabs.Tab>
