# AnimaXLoaderRequest

`AnimaXLoaderRequest` 描述待加载的资源信息，通常会传给 [AnimaXLoaderProtocol](./animax-loader-protocol)。

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

## 创建方式

### requestWithURL

使用 URL 创建加载请求。

**参数**

| 名称  | 类型         | 说明       |
| ----- | ------------ | ---------- |
| `url` | `NSString *` | 资源地址。 |

**返回值**

`AnimaXLoaderRequest *`

### requestWithURL:requestProperties

使用 URL 和附加参数创建加载请求。

**参数**

| 名称         | 类型             | 说明               |
| ------------ | ---------------- | ------------------ |
| `url`        | `NSString *`     | 资源地址。         |
| `properties` | `NSDictionary *` | 附加参数，可为空。 |

**返回值**

`AnimaXLoaderRequest *`

## 属性

| 名称                | 类型             | 说明               |
| ------------------- | ---------------- | ------------------ |
| `url`               | `NSString *`     | 资源地址。         |
| `requestProperties` | `NSDictionary *` | 附加参数，可为空。 |

## 请求信息

### getUri

返回资源地址。

**返回值**

`String`

### getParams

返回资源加载附加参数。

**返回值**

`Map<String, Object>`

### getImageInfo

返回图片尺寸信息。非图片请求可以返回 `null`。

**返回值**

`IImageInfo`

## IImageInfo

### getWidth

返回图片宽度。

**返回值**

`int`

### getHeight

返回图片高度。

**返回值**

`int`
