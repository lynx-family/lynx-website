# AnimaXLoaderResponse

`AnimaXLoaderResponse` 包含资源数据或错误信息。加载器完成后，通过 completion handler 将响应传回 AnimaX。

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

## 创建方式

### responseWithPayload

创建成功响应。`payload` 可传入文件路径、字节数据或图片等加载结果。

**参数**

| 名称      | 类型 | 说明                   |
| --------- | ---- | ---------------------- |
| `payload` | 泛型 | 加载成功时的资源数据。 |

**返回值**

`AnimaXLoaderResponse *`

### responseWithError

使用 `NSError` 创建失败响应。

**参数**

| 名称    | 类型        | 说明       |
| ------- | ----------- | ---------- |
| `error` | `NSError *` | 错误信息。 |

**返回值**

`AnimaXLoaderResponse *`

### responseWithErrorMessage

使用错误信息创建失败响应。

**参数**

| 名称           | 类型         | 说明       |
| -------------- | ------------ | ---------- |
| `errorMessage` | `NSString *` | 错误信息。 |

**返回值**

`AnimaXLoaderResponse *`

## 属性

| 名称      | 类型         | 说明                   |
| --------- | ------------ | ---------------------- |
| `payload` | 泛型 payload | 加载成功时的资源数据。 |
| `error`   | `NSError *`  | 加载失败时的错误信息。 |

## 创建方式

### createStringFilePathResponse

使用本地文件路径创建成功响应。

**参数**

| 名称       | 类型     | 说明         |
| ---------- | -------- | ------------ |
| `filePath` | `String` | 本地文件路径 |

**返回值**

`AnimaXLoaderResponse<String>`

### createByteArrayResponse

使用字节数组创建成功响应。

**参数**

| 名称    | 类型     | 说明       |
| ------- | -------- | ---------- |
| `bytes` | `byte[]` | 资源数据。 |

**返回值**

`AnimaXLoaderResponse<byte[]>`

### createBitmapResponse

使用 bitmap 引用创建成功响应。

**参数**

| 名称     | 类型                              | 说明          |
| -------- | --------------------------------- | ------------- |
| `bitmap` | `IAnimaXCloseableBitmapReference` | bitmap 引用。 |

**返回值**

`AnimaXLoaderResponse<IAnimaXCloseableBitmapReference>`

### createErrorResponse

使用 `Throwable` 创建失败响应。

**参数**

| 名称    | 类型        | 说明       |
| ------- | ----------- | ---------- |
| `error` | `Throwable` | 错误信息。 |

**返回值**

`AnimaXLoaderResponse<Throwable>`

## 响应数据

### getData

返回响应携带的数据。

**返回值**

`T`
