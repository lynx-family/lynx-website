# AnimaXLoaderProtocol

`AnimaXLoaderProtocol` 用于实现自定义资源加载能力。通常由 [AnimaXResourceFactoryService](./animax-resource-factory-service) 创建并返回。

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

## 加载请求

### handleRequest

执行加载请求。加载完成后，通过 [AnimaXLoaderCompletionHandler](./animax-loader-completion-handler) 将 [AnimaXLoaderResponse](./animax-loader-response) 回传给 AnimaX。

**参数**

| 名称         | 类型                                                                  | 说明       |
| ------------ | --------------------------------------------------------------------- | ---------- |
| `request`    | [`AnimaXLoaderRequest`](./animax-loader-request) `*`                  | 加载请求。 |
| `completion` | [`AnimaXLoaderCompletionHandler`](./animax-loader-completion-handler) | 完成回调。 |

## 资源来源

### getScheme

返回当前 loader 处理的资源来源。

**返回值**

`getScheme` 返回 [AnimaXLoaderScheme](./animax-loader-scheme)。

</PlatformTabs.Tab>

## 加载请求

### load

执行加载请求。加载完成后，通过 [AnimaXLoaderCompletionHandler](./animax-loader-completion-handler) 将 [AnimaXLoaderResponse](./animax-loader-response) 回传给 AnimaX。

**参数**

| 名称                | 类型                                                                  | 说明       |
| ------------------- | --------------------------------------------------------------------- | ---------- |
| `request`           | [`AnimaXLoaderRequest`](./animax-loader-request)                      | 加载请求。 |
| `completionHandler` | [`AnimaXLoaderCompletionHandler`](./animax-loader-completion-handler) | 完成回调。 |

## 资源来源

### getScheme

返回当前 loader 处理的资源来源。

**返回值**

`getScheme` 返回 [AnimaXLoaderScheme](./animax-loader-scheme)。
