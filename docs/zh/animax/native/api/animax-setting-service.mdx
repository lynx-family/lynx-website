# AnimaXSettingService

`AnimaXSettingService` 对应 Android 的 `IAnimaXSettingService`，用于让接入方动态提供 AnimaX 运行时配置值。

## 读取配置

### getValueByKey

获取指定 key 的配置。

**参数**

| 名称  | 类型     | 说明       |
| ----- | -------- | ---------- |
| `key` | `String` | 配置 key。 |

**返回值**

`AnimaXSettingValue`。未找到或读取失败时返回空值对象。

## 注册方式

```java
AnimaXServiceCenter.inst().registerService(
    IAnimaXSettingService.class,
    mySettingService);
```
