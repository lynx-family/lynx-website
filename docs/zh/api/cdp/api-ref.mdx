## JavaScript 运行时支持

本参考中的 runtime-domain 方法由 PrimJS inspector 实现。PrimJS 同时用于
Main-Thread Scripting (MTS) 和 Background-Thread Scripting (BTS)。

基于 V8 的调试委托给 V8 Inspector。请参阅 [V8 CDP
参考文档](https://chromedevtools.github.io/devtools-protocol/1-3)，了解 Lynx
采用的 V8 Inspector protocol 版本。本参考不单独列出 V8 方法。
