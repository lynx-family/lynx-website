`title-bar-view` 元素类似于 Web 中的 `app-region` 样式，允许开发者自定义窗口的可拖拽区域。

## 属性

### `moveable`

{' '}

<ClayWindowsOnly /> <ClayMacOSOnly />

```tsx
// @defaultValue: false
moveable?: boolean;
```

当设置为 true 时，拖拽 title bar view 可以移动窗口。
