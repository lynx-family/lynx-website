标题栏视图元素类似于网页中的应用区域样式，
允许开发者自定义窗口拖动区域。

## 属性

### `moveable`

<ClayWindowsOnly /> <ClayMacOSOnly />

```tsx
// @默认值: false
moveable?: boolean;
```

当设置为 true 时，标题栏视图在拖动时能够移动窗口。
