# `<markdown>`

<APISummary />

用于在 Lynx 视图内渲染 Markdown 内容。

## 使用指南

`<markdown>` 是一个扩展元件，用于将 Markdown 格式的文本渲染为原生视图。它支持丰富的样式定制、打字机动画、选择复制、图片插入等功能。

以下示例展示了 `<markdown>` 元件在不同场景下的应用。

## 属性

### `content`

```ts
content: string;
```

Markdown 源内容文本。

### `markdown-style`

```ts
'markdown-style'?: object;
```

Markdown 的样式配置对象，是 tag 到样式配置项的映射。支持的 tag 包括：`normalText`、`h1`-`h6`、`link`、`inlineCode`、`codeBlock`、`quote`、`orderedList`、`unorderedList`、`table`、`tableCell`、`split`、`typewriterCursor`、`truncation`、`listItem`、`tableHeader`、`image`、`quoteBorderLine`、`unorderedListMarker`、`orderedListNumber`、`imageCaption` 等。

### `animation-type`

```ts
'animation-type'?: 'none' | 'typewriter';
```

动画类型。设置为 `typewriter` 开启打字机效果，默认为 `none`。

### `animation-velocity`

```ts
'animation-velocity'?: number;
```

打字机动画速度，单位为字符数/秒，默认为 1。

### `text-maxline`

```ts
'text-maxline'?: number;
```

文本最大行数，超出后截断。

### `content-complete`

```ts
'content-complete'?: boolean;
```

标明当前 `content` 是否完整。为 `true` 时动画结束后隐藏光标，为 `false` 时光标一直显示。默认为 `true`。

### `typewriter-dynamic-height`

```ts
'typewriter-dynamic-height'?: boolean;
```

配置打字机效果是否动态改变元素高度，默认为 `false`。

### `text-selection`

```ts
'text-selection'?: boolean;
```

配置 Markdown 内容是否可选择，默认为 `false`。

### `text-mark-attachments`

```ts
'text-mark-attachments'?: MarkAttachment[];
```

用于在指定字符范围内添加高亮或下划线。支持多段重叠和渐变色。

### `content-range`

```ts
'content-range'?: number[];
```

配置 Markdown 内容的解析范围，默认为 `[0, INT32_MAX]`。

### `markdown-effect`

```ts
'markdown-effect'?: object;
```

配置额外的 Markdown 效果，如文本渐隐蒙层。

### `enable-gif`

```ts
'enable-gif'?: boolean;
```

配置 iOS 是否开启 GIF 动画支持，Android 默认支持。默认为 `false`。

### `initial-animation-step`

```ts
'initial-animation-step'?: number;
```

指定动画开始的位置，默认为 0。

### `typewriter-height-transition-duration`

```ts
'typewriter-height-transition-duration'?: number;
```

打字机高度变化时的渐变时间，单位为秒，默认为 0。

### `content-id`

```ts
'content-id'?: string;
```

作为 `content` 的唯一标识符，用于在事件中消费。

## 事件

### `binddrawStart`

```ts
binddrawStart?: (e: BaseEvent) => void;
```

Markdown 开始绘制时触发。

### `binddrawEnd`

```ts
binddrawEnd?: (e: BaseEvent) => void;
```

Markdown 结束绘制时触发。

### `bindanimationStep`

```ts
bindanimationStep?: (e: BaseEvent<'bindanimationStep', {
  animationStep: number;
  maxAnimationStep: number;
}>) => void;
```

每次动画帧的回调事件。

### `bindoverflow`

```ts
bindoverflow?: (e: BaseEvent<'bindoverflow', {
  type: 'ellipsis' | 'clip';
}>) => void;
```

文本溢出时触发。

### `bindlink`

```ts
bindlink?: (e: BaseEvent<'bindlink', {
  url: string;
  content: string;
}>) => void;
```

链接被点击时触发。

### `bindselectionchange`

```ts
bindselectionchange?: (e: BaseEvent<'bindselectionchange', {
  start: number;
  end: number;
}>) => void;
```

选中区域变化时触发。

### `bindimageTap`

```ts
bindimageTap?: (e: BaseEvent<'bindimageTap', {
  url: string;
}>) => void;
```

图片被点击时触发。

### `bindparseEnd`

```ts
bindparseEnd?: (e: BaseEvent) => void;
```

Markdown 内容解析完成时触发。

### `bindtextClick`

```ts
bindtextClick?: (e: BaseEvent<'bindtextClick', {
  text: string;
}>) => void;
```

文本被点击时触发。

## 方法

### `pauseAnimation`

暂停打字机动画。

### `resumeAnimation`

恢复打字机动画。

### `getContent`

获取当前渲染的内容。

### `getSelectedText`

获取当前选中区域的文本。

### `setTextSelection`

设置选中区域。参数包括 `startX`、`startY`、`endX`、`endY`，以及可选的 `selectionTextType`（`char` | `word` | `sentence` | `paragraph`）。

### `getTextBoundingRect`

获取指定范围文本的位置信息。参数为 `start` 和 `end`，返回 `TextBoundingRect` 对象。

### `getImages`

获取 Markdown 中所有图片的信息。

### `getParseResult`

获取 Markdown 的解析结果。
