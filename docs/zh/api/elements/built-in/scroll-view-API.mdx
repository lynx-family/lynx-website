## 事件

前端可以绑定相应的事件回调来监听元素的运行时行为，如下所示。

### `bindcontentsizechanged`

{' '}
<AndroidOnly /> <IOSOnly /> <HarmonyOnly /> <VersionBadge>1.6</VersionBadge>

```tsx
bindcontentsizechanged = (e: ContentSizeChangedEvent) => {};
```

当滚动视图的内容大小发生变化时，会触发此事件。

### `bindscroll`

{' '}
<AndroidOnly /> <IOSOnly /> <ClayOnly /> <HarmonyOnly /> <VersionBadge>
  1.4
</VersionBadge>

```tsx
bindscroll = (e: ScrollEvent) => {};
```

当滚动视图正在滚动时，会触发此事件。

### `bindscrollend`

{' '}
<AndroidOnly /> <IOSOnly /> <ClayOnly /> <HarmonyOnly /> <VersionBadge>
  1.6
</VersionBadge>

```tsx
bindscrollend = (e: ScrollEndEvent) => {};
```

当滚动视图的滚动结束时，会触发此事件。

### `bindscrolltolower`

{' '}
<AndroidOnly /> <IOSOnly /> <HarmonyOnly /> <VersionBadge>1.4</VersionBadge>

```tsx
bindscrolltolower = (e: ScrollToLowerEvent) => {};
```

当滚动区域的底部/右侧边缘与由 `lowerThreshold` 定义的可见区域相交时，会触发此事件。

### `bindscrolltoupper`

{' '}
<AndroidOnly /> <IOSOnly /> <HarmonyOnly /> <VersionBadge>1.4</VersionBadge>

```tsx
bindscrolltoupper = (e: ScrollToUpperEvent) => {};
```

当滚动区域的顶部/左侧边缘与由 `upperThreshold` 定义的可见区域相交时，会触发此事件。

## 方法

前端可以通过 [SelectorQuery](/api/lynx-api/nodes-ref/nodes-ref-invoke.html) API 调用组件方法。

### `autoScroll`

{' '}
<AndroidOnly /> <IOSOnly /> <ClayOnly /> <HarmonyOnly />
```ts

lynx.createSelectorQuery()
.select('#id')
.invoke({
method: 'autoScroll',
params: {
/**
_ 每秒滚动的距离，支持正负数。距离单位可以是 "px"、"rpx"、"ppx" 或空（对于 iOS，该值必须大于 1/屏幕缩放比 px）。
_ @Android
_ @iOS
_ @Harmony
_ @PC
_/
rate: number;
/**
_ 开始/停止自动滚动。
_ @Android
_ @iOS
_ @Harmony
_ @PC
_/
start: boolean;
};
success: function (res) {},
fail: function (res) {},
})
.exec();

````

自动滚动

### `getScrollInfo`

 <AndroidOnly /> <IOSOnly /> <ClayOnly /> <HarmonyOnly />
```ts

lynx.createSelectorQuery()
     .select('#id')
     .invoke({
      method: 'getScrollInfo',
      success: Callback<{
        /**
         * 沿方向的总可滚动范围，单位为 PX
         * @Android
         * @iOS
         * @Harmony
         * @PC
         */
        scrollRange: number;
        /**
         * 内容在 X 轴上的偏移量，单位为 PX
         * @Android
         * @iOS
         * @Harmony
         * @PC
         */
        scrollX: number;
        /**
         * 内容在 Y 轴上的偏移量，单位为 PX
         * @Android
         * @iOS
         * @Harmony
         * @PC
         */
        scrollY: number;
      }>;
      fail: function (res) {},
    })
    .exec();

````

获取滚动信息

### `scrollTo`

 <AndroidOnly /> <IOSOnly /> <ClayOnly /> <HarmonyOnly />
```ts

lynx.createSelectorQuery()
     .select('#id')
     .invoke({
      method: 'scrollTo',
      params: {
        /**
         * 目标项索引
         * @defaultValue 0
         */
        index?: number;
        /**
         * 相对于目标节点的偏移量
         */
        offset?: number;
        /**
         * 启用滚动动画
         */
        smooth?: boolean;
      };
      success: function (res) {},
      fail: function (res) {},
    })
    .exec();

````
