# `<frame>`

一个类似 HTML `<iframe>` 的页面元素，可以将一个 Lynx 页面迁入当前页面中。

:::tip
`<frame>` 元件于 Lynx 3.4 版本引入，需要使用该元件的场景，需将依赖的 Lynx 版本升级至 3.4 及以上。
:::

## 属性

:::caution
`<frame>` 暂不支持自动撑高，需要手动设置 frame 的宽高。
:::

### `src` <Required /> \{#required-src}

```ts
src: string;
```

设置 frame 资源的加载路径，路径指向的资源将通过 [TemplateResourceFetcher](../../lynx-native-api/resource-fetcher/TemplateResourceFetcher.mdx) 进行模板获取。

:::info

1. `<frame>` 加载的页面可以是 Lynx 任意页面，但暂不支持 [Lazy Bundle](/guide/spec.html#lazy-bundle)。
2. 若加载资源为本地离线资源，frame 页面可以与当前 Lynx 页面在同一帧中上屏显示。

:::

### `data`

```ts
data?: object;
```

向 frame 嵌套的 Lynx 页面传递数据
