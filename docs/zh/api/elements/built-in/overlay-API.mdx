## 属性

### `ios-enable-swipe-back`

{' '}
<IOSOnly />

```tsx
// @默认值: false
'ios-enable-swipe-back'?: boolean;
```

当覆盖层显示时，'true' 允许向右滑动以关闭当前页面，'false' 则不允许。

### `level`

{' '}
<AndroidOnly /> <IOSOnly /> <ClayOnly /> <HarmonyOnly />

```tsx
// @默认值: 1
level?: 4 | 2 | 1 | 3;
```

引入了层的概念，分为四个级别。层的值越大，越靠近底部。默认情况下为第一级。层按 1 到 4 的顺序排列。指定要显示的层，且不受显示顺序的影响。在每层内，按照“后进先出”的逻辑排列。覆盖层显示时不能动态调整层，只能在隐藏时进行调整。 \*

### `mode`

{' '}
<IOSOnly />

```tsx
// @默认值: 'window'
mode?: string;
```

指定覆盖层内容在 iOS 上所处的级别。window：挂载在窗口上，即应用的最顶层；top：挂载在 topViewController 上；page：挂载在 UINavigationController 上；others：在客户端中自定义类名。

### `visible`

{' '}
<AndroidOnly /> <IOSOnly /> <ClayOnly /> <HarmonyOnly />

```tsx
// @默认值: false
visible?: boolean;
```

控制覆盖层是否显示

## 事件

前端可以绑定相应的事件回调来监听元素的运行时行为，如下所示。

### `binddismissoverlay`

{' '}
<AndroidOnly /> <IOSOnly /> <ClayOnly /> <HarmonyOnly />

```tsx
binddismissoverlay = (e: BaseEvent) => {};
```

遮罩层隐藏时的回调。

### `binderror`

{' '}
<AndroidOnly /> <VersionBadge>2.18</VersionBadge>

```tsx
binderror = (e: OverlayErrorEvent) => {};
```

| 字段      | 类型   | 是否可选 | 默认值 | 平台            | 起始版本 | 描述                                                                                            |
| --------- | ------ | -------- | ------ | --------------- | -------- | ----------------------------------------------------------------------------------------------- |
| errorCode | 字符串 | 否       | –      | <AndroidOnly /> |          | 错误码：0 - 正常显示；非 0 - 无法显示，你可以尝试使用下面提到的预加载方案来适配容器解决此问题。 |
| errorMsg  | 字符串 | 否       | –      | <AndroidOnly /> |          | 错误信息                                                                                        |

触摸遮罩层时的回调

### `bindoverlaytouch`

{' '}
<AndroidOnly /> <IOSOnly /> <ClayOnly />


| 字段  | 类型              | 是否可选 | 默认值 | 平台                                     | 起始版本 | 描述                           |

| state | OverlayTouchState | 否       | –      | <AndroidOnly /> <IOSOnly /> <ClayOnly /> |          | 触摸状态                       |

| y     | 数字              | 否       | –      | <AndroidOnly /> <IOSOnly /> <ClayOnly /> |          | 相对于窗口的 y 坐标，单位为 px |
