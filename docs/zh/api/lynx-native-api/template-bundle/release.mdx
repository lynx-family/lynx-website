# release

<APISummary />

释放当前 `TemplateBundle` 对象持有的 `Native` 内存。执行 `release` 方法后，`TemplateBundle` 将变为 `inValid` 状态。

## 语法

### Android

```java
public void release();
```

### Desktop (C++)

Desktop C++ API 没有单独暴露 `release()` 方法。`LynxTemplateBundle` 会在析构函数中释放底层句柄：

```cpp
{
  lynx::pub::LynxTemplateBundle bundle(content, length, dtor, opaque);
}  // 在 ~LynxTemplateBundle() 中释放
```

### Desktop (C API)

```c
void lynx_template_bundle_release(lynx_template_bundle_t* bundle);
```

如果 bundle 已经被 `lynx_view_load_template(...)` 消费，释放可能由 view 接管；否则需要由调用方主动释放。
