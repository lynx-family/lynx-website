# getErrorMessage

当 `TemplateBundle` 是一个无效对象时，使用此方法获取模板解析期间发生的异常信息。

## 语法

### Android

```java
public String getErrorMessage();
```

#### 返回值

- 异常信息。

### iOS

```objc
- (NSString *_Nullable)errorMsg;
```

#### 返回值

- 异常信息，如果返回 `nil`，则证明 `LynxTemplateBundle` 正常。

### Harmony

```typescript
public getErrorMessage(): string;
```

#### 返回值

- 模板解析期间的异常信息。
