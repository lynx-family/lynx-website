# isElementBundleValid

TemplateBundle 是否包含一个有效的 ElementBundle。

## 语法

### Android

```java
public boolean isElementBundleValid();
```

#### 返回值

- 如果有效则为 True，否则为 false。

### iOS

```objc
- (BOOL)isElementBundleValid;
```

#### 返回值

- 如果 TemplateBundle 包含一个有效的 ElementBundle，则为 True，否则为 false。

### Harmony

```typescript
public isElementBundleValid(): boolean;
```

#### 返回值

- 如果有效则为 True，否则为 false。
