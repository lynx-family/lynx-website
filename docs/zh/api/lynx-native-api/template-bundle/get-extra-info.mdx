# getExtraInfo

读取前端模板 `pageConfig` 中配置的 `extraInfo` 字段的内容。

## 语法

### Android

```java
public Map< String, Object > getExtraInfo();
```

#### 返回值

- 当前端未配置 `extraInfo` 或在空的 `TemplateBundle` 对象上调用时，返回 `null`，否则返回 `extraInfo` 字段。

### iOS

```objc
- (NSDictionary *_Nullable)extraInfo;
```

#### 返回值

- 当前端未配置 `extraInfo` 或在空的 `TemplateBundle` 对象上调用时，返回 `nil`，否则返回 `extraInfo` 字段。

### Harmony

```typescript
public getExtraInfo(): Map<string, Object>;
```

#### 返回值

- 当前端未配置 extraInfo 或在空的 TemplateBundle 对象上调用时，返回 null。
