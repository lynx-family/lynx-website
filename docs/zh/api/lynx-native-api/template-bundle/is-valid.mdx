# isValid

<APISummary />

确定当前的 `TemplateBundle` 对象是否有效。

## 语法

### Android

```java
public boolean isValid();
```

#### 返回值

- 如果当前的 `TemplateBundle` 对象有效，则为 `true`，否则为 `false`。

### Desktop (C++)

```cpp
bool IsValid() const;
```

#### 返回值

- 当前 `LynxTemplateBundle` 对象有效时返回 `true`，否则返回 `false`。

### Desktop (C API)

```c
int lynx_template_bundle_is_valid(lynx_template_bundle_t* bundle);
```

#### 返回值

- 当前 `lynx_template_bundle_t` 对象有效时返回 `1`，否则返回 `0`。
