# fetchTemplate

`LynxEngine` 内部调用此方法来获取 `Bundle` 和 `Lazy Bundle` 的内容。返回结果类型可以包含 `byte[]` 或 `TemplateBundle`。

## 语法

### Android

```java
public abstract void fetchTemplate(LynxResourceRequest request, LynxResourceCallback< TemplateProviderResult > callback);
```

#### 参数说明

- `request`: 请求所需的内容文件。
- `callback`: 响应所需的内容文件：`byteArray` 或 `TemplateBundle`。

:::note

- 此方法必须实现。

:::

### iOS

```objc
- (void)fetchTemplate:(LynxResourceRequest *_Nonnull)request
           onComplete:(LynxTemplateResourceCompletionBlock _Nonnull)callback;
```

#### 参数说明

- `request`: 请求所需的内容文件。
- `callback`: 响应所需的内容文件：`byteArray` 或 `TemplateBundle`。

:::note

- 此方法必须实现。

:::

### Harmony

```typescript
abstract fetchTemplate(request: LynxResourceRequest, callback: AsyncCallback<TemplateProviderResult>): void;
```

#### 参数说明

- `request`: 获取 `Bundle` 和 `Lazy Bundle` 内容的请求。
- `callback`: 以 `ArrayBuffer` 类型响应所需的内容文件。
