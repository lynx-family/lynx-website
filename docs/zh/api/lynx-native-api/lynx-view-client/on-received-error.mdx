# onReceivedError

<APISummary />

收到错误时调用。

## 语法

### Android

```java
public void onReceivedError(LynxError error);
```

#### 参数

- `error`: 错误的类型和信息。

### Harmony

```typescript
public onReceivedError(error: LynxError): void;
```

#### 参数

- `error`: 错误的类型和信息。

### Desktop (C++)

```cpp
class YourViewClient : public lynx::pub::LynxViewClient {
 public:
  void OnReceivedError(int error_code, const char* message) override;
};
```

#### 参数说明

- `error_code`: 错误码。
- `message`: 错误信息。

### Desktop (C API)

```c
typedef void (*on_received_error)(lynx_view_client_t* client,
                                  int error_code,
                                  const char* message);
void lynx_view_client_bind_on_received_error(lynx_view_client_t* client,
                                             on_received_error callback);
```

#### 参数说明

- `client`: 接收回调的 `lynx_view_client_t` 实例。
- `error_code`: 错误码。
- `message`: 错误信息。
