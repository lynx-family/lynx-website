# onDestroy

<APISummary />

页面销毁后调用。

## 语法

### Android

```java
public void onDestroy();
```

:::note

- 回调在主线程中执行。

:::

### Harmony

```typescript
public onDestroy(): void;
```

### Desktop (C++)

```cpp
class YourViewClient : public lynx::pub::LynxViewClient {
 public:
  void OnDestroy() override;
};
```

### Desktop (C API)

```c
typedef void (*on_destroy)(lynx_view_client_t* client);
void lynx_view_client_bind_on_destroy(lynx_view_client_t* client,
                                      on_destroy callback);
```
