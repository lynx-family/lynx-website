# onPageStart

<APISummary />

页面开始加载时调用。

## 语法

### Android

```java
public void onPageStart(@Nullable String url);
```

#### 参数

- `url`: 页面 URL

### iOS

```objc
- (void)lynxViewDidStartLoading:(LynxView *)view;
```

#### 参数

- `view`: LynxView 实例。

:::note

- 每个内容加载请求都会调用一次此方法。

:::

### Harmony

```typescript
public onPageStarted(lynxContext: LynxContext, pipelineInfo: LynxPipelineInfo): void;
```

#### 参数

- `lynxContext`: 已开始加载的 LynxContext。
- `pipelineInfo`: 渲染流水线信息。

:::note

- 此方法将在 Lynx 主进程之前执行，因此请不要在此方法中执行过于复杂的逻辑。

:::

### Desktop (C++)

```cpp
class YourViewClient : public lynx::pub::LynxViewClient {
 public:
  void OnPageStart(const char* url) override;
};
```

#### 参数说明

- `url`: 页面 URL。

### Desktop (C API)

```c
typedef void (*on_page_start)(lynx_view_client_t* client, const char* url);
void lynx_view_client_bind_on_page_start(lynx_view_client_t* client,
                                         on_page_start callback);
```

#### 参数说明

- `client`: 接收回调的 `lynx_view_client_t` 实例。
- `url`: 页面 URL。
