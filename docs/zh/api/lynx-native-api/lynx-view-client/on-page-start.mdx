# onPageStart

页面开始加载时调用。

## 语法

### Android

```java
public void onPageStart(@Nullable String url);
```

#### 参数

- `url`: 页面 URL

### iOS

```objc
- (void)lynxViewDidStartLoading:(LynxView *)view;
```

#### 参数

- `view`: LynxView 实例。

:::note

- 每个内容加载请求都会调用一次此方法。

:::
