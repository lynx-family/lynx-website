# onReportComponentInfo

返回使用的组件标签名。

## 语法

### Android

```java
public void onReportComponentInfo(Set<String> mComponentSet);
```

#### 参数

- `mComponentSet`: 组件标签名集合。

### iOS

```objc
- (void)lynxView:(LynxView *)view didReportComponentInfo:(NSSet<NSString *> *)componentSet;
```

#### 参数

- `view`: LynxView 实例。
- `componentSet`: 组件标签名集合。
