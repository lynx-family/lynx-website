# onPageUpdate

<APISummary />

页面更新时调用。

## 语法

### Android

```java
public void onPageUpdate();
```

### iOS

```objc
- (void)lynxViewDidPageUpdate:(LynxView *)view;
```

#### 参数

- `view`: LynxView 实例。

### Harmony

```typescript
public onPageUpdate(): void;
```

### Desktop (C++)

```cpp
class YourViewClient : public lynx::pub::LynxViewClient {
 public:
  void OnPageUpdated() override;
};
```

### Desktop (C API)

```c
typedef void (*on_page_updated)(lynx_view_client_t* client);
void lynx_view_client_bind_on_page_updated(lynx_view_client_t* client,
                                           on_page_updated callback);
```
