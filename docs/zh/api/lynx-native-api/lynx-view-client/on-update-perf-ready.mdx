# onUpdatePerfReady

界面更新完成后，性能数据统计回调。

## 语法

### Android

```java
public void onUpdatePerfReady(LynxPerfMetric metric);
```

:::note

- 由于渲染线程的差异，回调的时机不固定，不应作为任何业务方的起点。回调在主线程中。

:::

### iOS

```objc
- (void)lynxView:(LynxView *)view didReceiveUpdatePerf:(LynxPerformance *)perf;
```

#### 参数

- `view`: LynxView 实例。

:::note

- 由于渲染线程的差异，回调的时机不固定，不应作为任何业务方的起点。回调在主线程中。

:::
