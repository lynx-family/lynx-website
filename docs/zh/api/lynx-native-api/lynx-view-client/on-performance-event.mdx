# onPerformanceEvent

通知客户端性能事件已发送。每当生成性能事件时，都会调用该方法，包括但不限于容器初始化、引擎渲染、渲染指标更新等。

## 语法

### Harmony

```typescript
public onPerformanceEvent(entry: PerformanceEntry): void;
```

#### 参数

- `entry`: 性能事件的 [PerformanceEntry](../../lynx-api/performance-api/performance-entry.mdx)。
