# onFlushFinish

在异步渲染模式下，当 UI 刷新结束时调用。

## 语法

### Android

```java
public void onFlushFinish(FlushInfo flushInfo);
```

#### 参数

- `flushInfo`: 包含刷新的开始和结束时间，以及是否是同步刷新。
