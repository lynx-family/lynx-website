# onLynxEvent

报告发送到前端的 Lynx 事件。

## 语法

### Android

```java
public void onLynxEvent(LynxEventDetail detail);
```

#### 参数

- `detail`: 将发送到前端的 LynxEvent，包括 eventName、lynxview 等。

### iOS

```objc
- (void)onLynxEvent:(LynxEventDetail *)event;
```

#### 参数

- `event`: 将发送到前端的 LynxEvent，包括 eventName、lynxview 等。
