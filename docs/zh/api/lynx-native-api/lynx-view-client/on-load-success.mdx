# onLoadSuccess

<APISummary />

页面加载完成时调用。

## 语法

### Android

```java
public void onLoadSuccess();
```

### iOS

```objc
- (void)lynxView:(LynxView *)view didLoadFinishedWithUrl:(NSString *)url;
```

#### 参数

- `view`: LynxView 实例。
- `url`: 页面 URL。

:::note

- 每个内容加载请求都会调用一次此方法。

:::

### Harmony

```typescript
public onLoadSuccess(url: string): void;
```

#### 参数

- `url`: 页面 URL。

### Desktop (C++)

```cpp
class YourViewClient : public lynx::pub::LynxViewClient {
 public:
  void OnLoadSuccess() override;
};
```

### Desktop (C API)

```c
typedef void (*on_load_success)(lynx_view_client_t* client);
void lynx_view_client_bind_on_load_success(lynx_view_client_t* client,
                                           on_load_success callback);
```
