# onTemplateBundleReady

模板解码后提供可重用的 TemplateBundle。

## 语法

### Android

```java
public void onTemplateBundleReady(@NonNull TemplateBundle bundle);
```

#### 参数

- `bundle`: 可重复使用的 TemplateBundle，它非空但可能无效。

:::note

- 此回调默认禁用，您可以通过 LynxLoadMeta 中的 DUMP_ELEMENT 选项或 RECYCLE_TEMPLATE_BUNDLE 选项启用它。

:::

### iOS

```objc
- (void)onTemplateBundleReady:(LynxTemplateBundle *)bundle;
```

#### 参数

- `bundle`: 可重复使用的 TemplateBundle，它非空但可能无效。

:::note

- 此回调默认禁用，您可以通过 LynxLoadMeta 中的 enableRecycleTemplateBundle 选项启用它。

:::

### Harmony

```typescript
public onTemplateBundleReady(bundle: TemplateBundle): void;
```

#### 参数

- `bundle`: 可重复使用的 TemplateBundle，它非空但可能无效。

:::note

- 此回调默认禁用，您可以通过 LynxLoadMeta 中的 DUMP_ELEMENT 选项或 RECYCLE_TEMPLATE_BUNDLE 选项启用它。

:::
