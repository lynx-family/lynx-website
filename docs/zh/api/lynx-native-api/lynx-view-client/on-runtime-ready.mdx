# onRuntimeReady

JS 环境准备完成时调用。

## 语法

### Android

```java
public void onRuntimeReady();
```

:::note

- 回调在异步线程中。

:::

### iOS

```objc
- (void)lynxViewDidConstructJSRuntime:(LynxView *)view;
```

#### 参数

- `view`: LynxView 实例。
