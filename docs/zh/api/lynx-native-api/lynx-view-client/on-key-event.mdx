# onKeyEvent

报告所有 Android 按键事件，并带有指示是否已处理的标志。

## 语法

### Android

```java
public void onKeyEvent(KeyEvent event, boolean handled);
```

#### 参数

- `event`: Android 按键事件。
- `handled`: 指示事件是否已被 Lynx 处理（消费）。
