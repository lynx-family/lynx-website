# onFirstLoadPerfReady

首次加载完成后，性能数据统计回调。

## 语法

### Android

```java
public void onFirstLoadPerfReady(LynxPerfMetric metric);
```

:::note

- 由于渲染线程的差异，回调的时机不固定，不应作为任何业务方的起点。回调在主线程中。

:::
