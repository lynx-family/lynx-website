# onTASMFinishedByNative

在 ui 或 most_on_tasm 模式下，当原生布局完成时，或在 multi_thread 模式下，当 diff 完成时调用。

## 语法

### Android

```java
public void onTASMFinishedByNative();
```

### iOS

```objc
- (void)lynxViewOnTasmFinishByNative:(LynxView *)view;
```

#### 参数

- `view`: LynxView 实例。
