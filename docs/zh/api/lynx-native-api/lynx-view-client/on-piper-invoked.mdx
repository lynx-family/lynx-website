# onPiperInvoked

当 JSBridge 被调用时调用。

## 语法

### Android

```java
public void onPiperInvoked(Map<String, Object> info);
```

#### 参数

- `info`: JSBridge 的信息。`url: String`: Lynx 的 url；`module-name: String`: 模块名称；`method-name: String`: 方法名称；`params: List<Object>`: (可选) 其他必要参数。

### iOS

```objc
- (void)onPiperInvoked:(NSDictionary *)info;
```

#### 参数

- `info`: Piper 的信息。`url: NSString`: Lynx 的 url；`module-name: NSString`: 模块名称；`method-name: NSString`: 方法名称；`params: NSArray`: (可选) 其他必要参数。
