# shouldRedirectUrl

`LynxEngine` 通过内部调用此方法来重定向图片路径，要求返回结果为 `String` 类型。

## 语法

### Android

```java
public abstract String shouldRedirectUrl(LynxResourceRequest request);
```

#### 参数说明

- `request`: 资源请求。
- `callback`: 目标 URL。

:::note

- 此方法必须实现。

:::

### iOS

```objc
- (NSString *_Nonnull)shouldRedirectUrl:(LynxResourceRequest *_Nonnull)request;
```

#### 参数说明

- `request`: 资源请求。
- `callback`: 目标 URL。

:::note

- 此方法必须实现。

:::

### Harmony

```typescript
abstract shouldRedirectUrl(request: LynxResourceRequest): string;
```

#### 参数说明

- `request`: 需要重定向的请求。

#### 返回值
- 目标 URL。
