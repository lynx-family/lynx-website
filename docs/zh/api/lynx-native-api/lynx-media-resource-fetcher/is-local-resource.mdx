# isLocalResource

`LynxEngine` 内部会调用此方法来判断资源路径是否存在于磁盘中。

## 语法

### Android

```java
public OptionalBool isLocalResource(String url);
```

#### 参数说明

- `url`: 输入路径。

#### 返回值

- 如果是本地路径，则返回 `TRUE`；如果不是本地路径，则返回 `FALSE`；如果不确定，则返回 `UNDEFINED`。

:::note

- 此方法为可选实现。

:::

### iOS

```objc
- (LynxResourceOptionalBool)isLocalResource:(NSURL *_Nonnull)url;
```

#### 参数说明

- `url`: 输入路径。

#### 返回值
- 如果是本地路径，则返回 `TRUE`；如果不是本地路径，则返回 `FALSE`；如果不确定，则返回 `UNDEFINED`。

:::note

- 此方法为可选实现。

:::

### Harmony

```typescript
isLocalResource(url: string): LynxOptionalBool;
```

#### 参数说明

- `url`: 输入路径。

#### 返回值
- 如果是本地路径，则返回 `TRUE`；如果不是本地路径，则返回 `FALSE`；如果不确定，则返回 `UNDEFINED`。
