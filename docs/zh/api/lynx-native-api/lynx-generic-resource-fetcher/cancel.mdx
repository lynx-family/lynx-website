# cancel

取消请求所需的资源。

## 语法

### Android

```java
public void cancel(LynxResourceRequest request);
```

#### 参数说明

- `request`: 必需的请求。

:::note

- 这个方法是可选实现的。

:::

### Harmony

```typescript
abstract cancel(request: LynxResourceRequest): void;
```

#### 参数说明

- `request`: 要取消的请求。
