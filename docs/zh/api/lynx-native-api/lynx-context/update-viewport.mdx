# updateViewport

客户端更新 `LynxView` 视图大小。

## 语法

### Harmony

```typescript
public updateViewport(width: number, widthMode: number, height: number, heightMode: number): void;
```

#### 参数说明

- `width`: 当前 `LynxView` 宽度。
- `widthMode`: 宽度的测量模式。 0: Indefinite, 1: Definite, 2: AtMost.
- `height`: 当前 `LynxView` 高度。
- `heightMode`: 高度的测量模式。 0: Indefinite, 1: Definite, 2: AtMost.
