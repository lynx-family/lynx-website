# updateMetaData

使用 [LynxUpdateMeta](/api/lynx-native-api/lynx-update-meta) 来更新 `LynxView`，它是客户端更新模板数据的主要入口。

## 语法

### Harmony

```typescript
updateMetaData(data: MetaData): void;
```

#### 参数说明

- `data`: Lynx 模板更新元数据，开发者使用此数据结构来指定更新模板的可选数据内容。
