# sendGlobalEvent

通过客户端向前端发送全局事件，前端可以通过 `GlobalEventEmitter` 监听到该事件。

## 语法

### Harmony

```typescript
sendGlobalEvent(name: string, params: Object[]): void;
```

#### 参数说明

- `name`: 全局事件名称。
- `params`: 全局事件携带的参数。
