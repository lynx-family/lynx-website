# findViewByIdSelector

根据 `idSelector` 属性查找 `UIView`。

## 语法

### Android

```java
public View findViewByIdSelector(String id);
```

#### 参数说明

- `id`: 前端在布局文件中指定的视图的 id 属性。

#### 返回值

- 与 idSelector 属性相对应的 `UIView` 节点。

### iOS

```objc
- (nullable UIView *)viewWithIdSelector:(nonnull NSString *)idSelector;
```

#### 参数说明

- `idSelector`: 前端在布局文件中指定的视图的 id 属性。

#### 返回值
- 与 idSelector 属性相对应的 `UIView` 节点。
