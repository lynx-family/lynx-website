# updateMetaData

使用 [LynxUpdateMeta](/api/lynx-native-api/lynx-update-meta) 更新 `LynxView`，是客户端更新模板数据的主入口。

## 语法

### Android

```java
public void updateMetaData(LynxUpdateMeta meta);
```

#### 参数说明

- `meta`: Lynx 模板更新元数据，开发者通过该数据结构指定更新模板的可选数据内容。

### iOS

```objc
- (void)updateMetaData:(nonnull LynxUpdateMeta *)meta;
```

#### 参数说明

- `meta`: Lynx 模板更新元数据，开发者通过该数据结构指定更新模板的可选数据内容。

### Web

```ts
export type Cloneable<T = string | number | null | boolean | undefined> =
  | T
  | Record<string, T>
  | T[];

updateData(
  data: Cloneable,
  updateDataType: UpdateDataType,
  callback?: () => void,
): void;
updateGlobalProps(data: Cloneable): void;
```

#### 参数说明

- `data`: 要更新的数据，应与 `structuredClone` 兼容。
