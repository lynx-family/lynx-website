# findViewByName

根据 name 属性查找 `UIView`。

## 语法

### Android

```java
public View findViewByName(String name);
```

#### 参数说明

- `name`: 前端在布局文件中指定的视图的 name 属性。

#### 返回值

- 与 name 属性相对应的 `UIView` 节点。

### iOS

```objc
- (nullable UIView *)findViewWithName:(nonnull NSString *)name;
```

#### 参数说明

- `name`: 前端在布局文件中指定的视图的 name 属性。

#### 返回值
- 与 name 属性相对应的 `UIView` 节点。
