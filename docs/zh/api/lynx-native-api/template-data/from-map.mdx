# fromMap

以键值格式输入数据，并返回解析后的 `TemplateData` 对象。

## 语法

### Android

```java
public static TemplateData fromMap(Map<String, Object> data);
```

#### 参数说明

- `data`: `Map` 类型数据。

#### 返回值

- 由 `data` 构建的 `TemplateData`。

### iOS

```objc
- (instancetype)initWithDictionary:(NSDictionary *)dictionary;
```

#### 参数说明

- `dictionary`: `NSDictionary*` 类型数据。

#### 返回值

- 由 `data` 构建的 `TemplateData`。
