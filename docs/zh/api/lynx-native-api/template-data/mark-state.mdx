# markState

用关联的 dataProcessor 名称标记当前的 TemplateData。当此 TemplateData 在 [UpdateData](/api/lynx-native-api/lynx-view/update-meta-data) 中使用时，将根据此名称找到相应的 dataProcessor 进行数据预处理。

## 语法

### Android

```java
public void markState(String name);
```

#### 参数说明

- `name`: 标记的名称。

### iOS

```objc
- (void)markState:(NSString *)name;
```

#### 参数说明

- `name`: 标记的名称。
