# constructor

以字符串、键值或 json 格式输入数据，并返回解析后的 `TemplateData` 对象。

## 语法

### Harmony

```typescript
constructor(input: string | Record<string, Object> | Object, processor: string);
```

#### 参数说明

- `input`: 字符串、键值或 json 类型的数据。
- `processor`: 数据处理器。
