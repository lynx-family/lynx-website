# lynx_value_create_uint32

## 介绍

用给定的 uint32_t 值创建一个 bool 类型的 lynx_value

## 语法

```c
lynx_api_status lynx_value_create_uint32(lynx_api_env env,
                                         uint32_t value,
                                         lynx_value* result);
```

### 参数

| Key           | Description                                                    |
|:--------------|:---------------------------------------------------------------|
| [in] env      | lynx_value 的运行环境，一般情况下是 nullptr   |
| [in] value    | Unsigned 32-bit integer value to be represented in lynx_value. |
| [out] result  | 一个 int32_t 类型的 lynx_value                         |

### 返回值

Returns `lynx_api_ok` if the API succeeded.

## 示例

```c
lynx_value val;
lynx_api_status status = lynx_value_create_uint32(env, 10, &val);
if (status != lynx_api_ok) {
  // handle error.
}
```
