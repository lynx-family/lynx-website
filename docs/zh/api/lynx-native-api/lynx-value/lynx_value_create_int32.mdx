# lynx_value_create_int32

## 介绍

用给定的 double 值创建一个 int32_t 类型的 lynx_value

## 语法

```c
lynx_api_status lynx_value_create_int32(lynx_api_env env,
                                        int32_t value,
                                        lynx_value* result);
```

### 参数

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | lynx_value 的运行环境，一般情况下是 nullptr                      |
| [in] value    | uint32_t value 的值                                            |
| [out] result  | 一个 uint32_t 类型的 lynx_value                                 |

### 返回值

Returns `lynx_api_ok` if the API succeeded.

## 示例

```c
lynx_value val;
lynx_api_status status = lynx_value_create_int32(env, 0, &val);
if (status != lynx_api_ok) {
  // handle error.
}
```
