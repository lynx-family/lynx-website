# lynx_value_get_bool

## 介绍

从 lynx_value 中读取 bool 值

## 语法

```c
lynx_api_status lynx_value_get_bool(lynx_api_env env,
                                    lynx_value value,
                                    bool* result);
```

### 参数

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | lynx_value 的运行环境，一般情况下是 nullptr |
| [in] value    | bool 类型的 lynx_value                              |
| [out] result  | 读取的 bool 值                                            |

### 返回值

Returns `lynx_api_ok` if the API succeeded. Returns `lynx_api_bool_expected` if the types do not match.

## 示例

```c
bool ret;
lynx_api_status status =  lynx_value_get_bool(env, value, &ret);
if (status != lynx_api_ok) {
  // handle error.
}
```
