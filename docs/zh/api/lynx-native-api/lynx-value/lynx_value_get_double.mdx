# lynx_value_get_double

## 介绍

从 lynx_value 中读取 double 值

## 语法

```c
lynx_api_status lynx_value_get_double(lynx_api_env env,
                                      lynx_value value,
                                      double* result);
```

### 参数

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | lynx_value 的运行环境，一般情况下是 nullptr |
| [in] value    | double 类型的 lynx_value                            |
| [out] result  | 读取的 double 值                                            |

### 返回值

Returns `lynx_api_ok` if the API succeeded. Returns `lynx_api_double_expected` if the types do not match.

## 示例

```c
double ret;
lynx_api_status status =  lynx_value_get_double(env, value, &ret);
if (status != lynx_api_ok) {
  // handle error.
}
```
