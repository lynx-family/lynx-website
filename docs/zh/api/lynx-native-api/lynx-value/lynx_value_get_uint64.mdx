# lynx_value_get_uint64

## 介绍

Read the uint64_t value from lynx_value. 

## 语法

```c
lynx_api_status lynx_value_get_uint64(lynx_api_env env,
                                      lynx_value value,
                                      uint64* result);
```

### 参数

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | lynx_value 的运行环境，一般情况下是 nullptr |
| [in] value    | uint64_t 类型的 lynx_value                              |
| [out] result  | 读取的 uint64_t 值                         |

### 返回值

Returns `lynx_api_ok` if the API succeeded. Returns `lynx_api_uint64_expected` if the types do not match.

## 示例

```c
uint64_t ret;
lynx_api_status status = lynx_value_get_uint64(env, value, &ret);
if (status != lynx_api_ok) {
  // handle error.
}
```
