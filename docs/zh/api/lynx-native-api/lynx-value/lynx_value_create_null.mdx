# lynx_value_create_null

## 介绍

创建一个 null 类型的 lynx_value.

## 语法

```c
lynx_api_status lynx_value_create_null(lynx_api_env env,
                                       lynx_value* result);
```

### 参数

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | lynx_value 的运行环境，一般情况下是 nullptr |
| [out] result  | A lynx_value representing a null.                            |

### 返回值

Returns `lynx_api_ok` if the API succeeded.

## 示例

```c
lynx_value val;
lynx_api_status status = lynx_value_create_null(env, &val);
if (status != lynx_api_ok) {
  // handle error.
}
```
