# lynx_value_remove_reference

## 介绍

给引用类型的 lynx_value 引用计数减 1

## 语法

```c
lynx_api_status lynx_value_remove_reference(
    lynx_api_env env, lynx_value value, lynx_value_ref ref);
```

### 参数

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | lynx_value 的运行环境，一般情况下是 nullptr |
| [in] value    | lynx_value.                                                  |
| [out] ref     | 保留字段，直接传 NULL.                        |

### 返回值

Returns `lynx_api_ok` if the API succeeded.

## 示例

```c
lynx_value val;
lynx_api_status status = lynx_value_create_array(env, &val);
if (status != lynx_api_ok) {
  // handle error.
}
// do something.
lynx_value_remove_reference(env, val, NULL);
```
