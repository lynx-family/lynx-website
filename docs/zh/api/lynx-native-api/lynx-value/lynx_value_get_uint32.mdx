# lynx_value_get_uint32

## 介绍

Read the uint32_t value from lynx_value. 

## 语法

```c
lynx_api_status lynx_value_get_uint32(lynx_api_env env,
                                      lynx_value value,
                                      uint32* result);
```

### 参数

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | lynx_value 的运行环境，一般情况下是 nullptr |
| [in] value    | uint32_t 类型的 lynx_value                            |
| [out] result  | 读取的 uint32_t 值                       |

### 返回值

Returns `lynx_api_ok` if the API succeeded. Returns `lynx_api_uint32_expected` if the types do not match.

## 示例

```c
uint32_t ret;
lynx_api_status status = lynx_value_get_uint32(env, value, &ret);
if (status != lynx_api_ok) {
  // handle error.
}
```
