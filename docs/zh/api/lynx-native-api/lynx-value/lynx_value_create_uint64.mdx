# lynx_value_create_uint64

## 介绍

用给定的 uint64_t 值创建一个 bool 类型的 lynx_value

## 语法

```c
lynx_api_status lynx_value_create_uint64(lynx_api_env env,
                                         uint64_t value,
                                         lynx_value* result);
```

### 参数

| Key           | Description                                                    |
|:--------------|:---------------------------------------------------------------|
| [in] env      | lynx_value 的运行环境，一般情况下是 nullptr   |
| [in] value    | uint64_t value 的值      |
| [out] result  | 一个 int64_t 类型的 lynx_value                           |

### 返回值

Returns `lynx_api_ok` if the API succeeded.

## 示例

```c
lynx_value val;
lynx_api_status status = lynx_value_create_uint64(env, 1000, &val);
if (status != lynx_api_ok) {
  // handle error.
}
```
