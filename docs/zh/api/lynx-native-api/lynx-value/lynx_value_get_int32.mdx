# lynx_value_get_int32

## 介绍

从 lynx_value 中读取 int32_t 值

## 语法

```c
lynx_api_status lynx_value_get_int32(lynx_api_env env,
                                     lynx_value value,
                                     int32* result);
```

### 参数

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | lynx_value 的运行环境，一般情况下是 nullptr |
| [in] value    | int32_t 类型的 lynx_value                               |
| [out] result  | 读取的 int32_t 值                                   |

### 返回值

Returns `lynx_api_ok` if the API succeeded. Returns `lynx_api_int32_expected` if the types do not match.

## 示例

```c
int32_t ret;
lynx_api_status status = lynx_value_get_int32(env, value, &ret);
if (status != lynx_api_ok) {
  // handle error.
}
```
