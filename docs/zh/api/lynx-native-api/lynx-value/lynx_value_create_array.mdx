# lynx_value_create_array

## 介绍

创建一个数组类型的 lynx_value.

## 语法

```c
lynx_api_status lynx_value_create_array(lynx_api_env env,
                                        lynx_value* result);
```

### 参数

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | lynx_value 的运行环境，一般情况下是 nullptr                      |
| [out] result  | 一个数组类型的 lynx_value.                                     |

### 返回值

Returns `lynx_api_ok` if the API succeeded.

## 示例

数组在堆上创建，使用完后需要用 `lynx_value_remove_reference` 释放内存

```c
lynx_value val;
lynx_api_status status = lynx_value_create_array(env, &val);
if (status != lynx_api_ok) {
  // handle error.
}
// do something.
lynx_value_remove_reference(env, val, NULL);

```
