# lynx_value_create_double

## 介绍

用给定的 double 值创建一个 bool 类型的 lynx_value

## 语法

```c
lynx_api_status lynx_value_create_double(lynx_api_env env,
                                         double value,
                                         lynx_value* result);
```

### 参数

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | lynx_value 的运行环境，一般情况下是 nullptr                      |
| [in] value    | double Value 的值                                             |
| [out] result  | 一个 double 类型的 lynx_value                                  |

### 返回值

Returns `lynx_api_ok` if the API succeeded.

## 示例

```c
lynx_value val;
lynx_api_status status = lynx_value_create_double(env, 1.1f, &val);
if (status != lynx_api_ok) {
  // handle error.
}
```
