# setLynxInspectorConsoleDelegate

<APISummary />

注册 `LynxView` 实例的 Console 监听器，用于监听该实例的所有 Console 消息。

## 语法

### iOS

```objective-c title="LynxBaseInspectorOwner.h"
- (void)setLynxInspectorConsoleDelegate:(id _Nonnull)delegate;
```

### Android

```java title="LynxBaseInspectorOwner.java"
void setLynxInspectorConsoleDelegate(Object delegate);
```

### Harmony

```ts title="LynxBaseInspectorOwner.ets"
setLynxInspectorConsoleDelegate(delegate: LynxInspectorConsoleDelegate): void;
```

### 参数说明

- `delegate`: 实现了 [`LynxInspectorConsoleDelegate`](/api/lynx-devtool-native-api/lynx-inspector-console-delegate#语法) 的对象。

## 使用示例

请参考 [LynxInspectorConsoleDelegate](/api/lynx-devtool-native-api/lynx-inspector-console-delegate#使用示例)。
