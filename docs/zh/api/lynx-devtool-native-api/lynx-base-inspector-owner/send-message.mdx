# sendMessage

<APISummary />

向通信对端（如 [DevTool 桌面应用](/guide/devtool#在开发设备上运行-devtool-桌面应用)）发送一条指定类型的调试消息。

配合 [`subscribeMessage`](/api/lynx-devtool-native-api/lynx-base-inspector-owner/subscribe-message) 方法，支持双向通信。

## 语法

### iOS

```objective-c title="LynxBaseInspectorOwnerNG.h"
- (void)sendMessage:(nonnull CustomizedMessage *)message;
```

#### 参数说明

- `message`：要发送的消息，[`CustomizedMessage`](/api/lynx-devtool-native-api/customized-message) 类型。

### Android

```java title="LynxBaseInspectorOwnerNG.java"
void sendMessage(CustomizedMessage message);
```

#### 参数说明

- `message`：要发送的消息，[`CustomizedMessage`](/api/lynx-devtool-native-api/customized-message) 类型。

### Harmony

```ts title="LynxBaseInspectorOwner.ets"
sendMessage(type: string, message: string): void;
```

#### 参数说明

- `type`：要发送的消息类型，如 [CDP](https://chromedevtools.github.io/devtools-protocol/) 或其他自定义消息类型。
- `message`：要发送的消息数据。

## 使用示例

请参考 [`subscribeMessage`](/api/lynx-devtool-native-api/lynx-base-inspector-owner/subscribe-message#使用示例)。
