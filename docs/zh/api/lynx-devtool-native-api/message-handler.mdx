# MessageHandler

<APISummary />

一个**协议（或接口）**，作为调试消息的处理器。

关于如何订阅调试消息，请参考 [`subscribeMessage`](/api/lynx-devtool-native-api/lynx-base-inspector-owner/subscribe-message)。

## 语法

实现 `MessageHandler` 协议（或接口），通过 `onMessage` 方法接收调试消息。

### iOS

```objective-c title="LynxBaseInspectorOwnerNG.h"
@protocol MessageHandler <NSObject>

@required
- (void)onMessage:(nonnull NSString *)message;

@end
```

### Android

```java title="MessageHandler.java"
public interface MessageHandler {
  void onMessage(String message);
}
```

### Harmony

```ts title="MessageHandler.ets"
export interface MessageHandler {
  onMessage: (msg: string) => void;
}
```
