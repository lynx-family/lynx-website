# setDevToolLibraryLoader

<APISummary />

为 DevTool 设置自定义的库加载器，用于在运行时加载 DevTool 所需的动态链接库（例如，`liblynxdevtool.so`）。

如果不调用此方法，DevTool 会使用默认的库加载器。

## 语法

### Android

```java title="LynxDevToolUtils.java"
static public void setDevToolLibraryLoader(INativeLibraryLoader loader);
```

#### 参数说明

- `loader`：一个实现了 `INativeLibraryLoader` 接口的自定义库加载器实例。
