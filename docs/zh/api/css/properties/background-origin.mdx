# background-origin

## 介绍

`background-origin` 规定了指定背景图片 [`background-image`](./background-image.mdx) 属性的原点位置的背景相对区域。

> 注意：当使用 [`background-color`](./background-color.mdx) 时，此属性无效。

## 语法

```css
background-origin: border-box;
background-origin: padding-box;
background-origin: content-box;

background-origin: content-box, padding-box;
```

### 取值

- 默认值 **padding-box** <br/>
  背景图片的摆放以 padding 区域为参考

- **border-box** <br/>
  背景图片的摆放以 border 区域为参考

- **content-box** <br/>
  背景图片的摆放以 content 区域为参考

## 形式语法

```
border-box | padding-box | content-box
```

## 与 Web 的区别

- 不支持 `inherit` 模式
- 对 `background-color` 不生效
