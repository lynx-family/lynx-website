# outline-width

## 介绍

用于设置轮廓的宽度。

## 使用示例

![demo](https://lf-lynx.tiktok-cdns.com/obj/lynx-artifacts-oss-sg/lynx-website/assets/doc/IMG_3703.jpeg)

### 取值

- `thin`
  细边框。

- **`medium`**
  中等边框。

- `thick`
  粗边框。

- [`<length>`](api/css/data-type/length.mdx)
  以具体的尺寸定义边框宽度

## 形式语法

```
thin | medium | thick | <length>
```

样例：

```css
outline-width: thin;
outline-width: thick;
outline-width: 10px;
outline-width: 20px;
```

## 与 Web 的区别

- [mdn:outline-width](https://developer.mozilla.org/en-US/docs/Web/CSS/outline-width)

不支持 inherit、initial、revert、unset 等
