# word-break

## 介绍

用于指定怎样在单词内断行。

## 语法

```css
word-break: break-all;
word-break: normal;
```

### 取值

- `normal`
  **默认值**。使用默认的断行规则。

- `break-all`
  可以在任意字符间断行。

- `keep-all`
  CJK 文本不断行。

- 默认值
  **normal**
