# animation-delay

## 介绍

`animation-delay` 属性指示动画起始播放延迟。

## 使用示例

  example="css-api"

## 语法

```css
animation-delay: 3s;
animation-delay: 2s, 4ms;
```

## 注意事项

:::caution

- Lynx 默认 `duration` 和 `delay` 需要**有单位**，无单位的写法可能会产生未定义的行为。

:::
