## 介绍

`text-stroke` 属性指定了文字的笔画宽度和颜色。

## 语法

```css
text-stroke: 4px green;
```

### 取值

- none <br/>
  **默认值**

- [`<length>`](/api/css/data-type/length.mdx)<br/>
  设置笔画的宽度

- [`<color>`](./color.mdx)<br/>
  设置描边的颜色

## 形式语法

```
none | <length[0,∞]> || <color>
```

## 与 Web 的区别

- 不支持设置笔画宽度为 thin 等属性值。

## 注意

某些字体可能有重叠的笔画。
