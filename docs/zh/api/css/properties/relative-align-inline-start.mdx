# relative-align-inline-start

## 介绍

指定当前元件与父亲或者对应 [`id`](./relative-id.mdx) 的同级元件的左/右边缘对齐。

当 [`direction`](./direction.mdx) 为 `ltr` 时，`relative-align-inline-start` 等价于 [`relative-align-left`](./relative-align-left.mdx)；

当 [`direction`](./direction.mdx) 为 `rtl/lynx-rtl` 时，`relative-align-inline-start` 等价于 [`relative-align-right`](./relative-align-right.mdx)；

## 语法

```css
relative-align-inline-start: parent;
relative-align-inline-start: none;

relative-align-inline-start: 1;
relative-align-inline-start: 2;
```

### 取值

`relative-align-inline-start` 属性的取值与 [`relative-align-left`](./relative-align-left) 属性相同。

```

```

## 与 Web 的区别

- Web 无此属性。
