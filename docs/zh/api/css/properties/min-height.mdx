# min-height

## 介绍

设置元件的最小高度。它可以阻止 [`height`](./height.mdx) 属性的应用值小于 `min-height` 的值。
`min-height` 的值会覆盖 [`height`](./height.mdx)，但是会被 [`max-height`](./max-height.mdx) 覆盖。

## 语法

```css
/* <length> */
min-height: 120px;
min-height: 10em;

/* <percentage> */
min-height: 75%;
```

### 取值

#### `0px`

**默认值**。

#### `<length>`

[`<length>`](api/css/data-type/length.mdx) 定义了一个确定值。

#### `<percentage>`

定义 `min-height` 为一个相对于父容器高度的百分数。

## 形式语法

```
min-height = <length-percentage [0,∞]>

<length-percentage> =
  <length>      |
  <percentage>
```

## 与 Web 的区别

[mdn: min-height](https://developer.mozilla.org/en-US/docs/Web/CSS/min-height)

1. 不支持 `auto`，默认值是 `0px`。
2. 不支持 `none`，`max-content`，`min-content`，`fit-content`。
3. 在 Lynx 中 `min-height` 不会覆盖 [`max-height`](./min-height.mdx)。
