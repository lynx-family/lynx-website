# font-size

## 介绍

`font-size` 属性指定字体的大小。因为该属性的值会被用于计算 `em` 和 `rem` 的长度单位。定义该值可能会改变其他元件的大小。

## 语法

```css
/* <length> */
font-size: 12px;
font-size: 0.8em;

/* <percentage> */
font-size: 80%;
```

### 取值

- [`<length>`](api/css/data-type/length.mdx) <br/>
  只能是正数。当单位为 `em` 或 `rem` 时，大小相对与父元件或跟元件的字体大小。

- [`<percentage>`](api/css/data-type/percentage.mdx) <br/>
  相对父元件字体大小。

## 形式语法

```
font-size = <length> | <percentage>
```

## 与 Web 的区别

- 不支持 `xx-small`、`x-small` 等指定绝对字体大小的关键字。

- 默认值是 `14px`，web 中默认值是 `medium` 指定的绝对大小。
