# animation-duration

## 介绍

`animation-duration` 用于设置动画持续的时间。

## 使用示例

<Go
  example="css-api"
  img="https://lf-lynx.tiktok-cdns.com/obj/lynx-artifacts-oss-sg/lynx-website/assets/doc/css/animation-duration.gif"

## 语法

```css
/* Single animation */
animation-duration: 6s
animation-duration: 120ms

/* Multiple animations */
animation-duration: 1s, 15s
animation-duration: 10s, 30s, 230ms
```

## 形式定义

import { PropertyDefinition } from '@/components/PropertyDefinition';

<PropertyDefinition
  initialValue={<>0s</>}
  appliesTo={<>all elements</>}
  inherited="no"
  animatable="no"
/>

## 形式语法

```css
/*默认值0s*/
animation-duration: <time>;
```

## 注意事项

:::caution

- Lynx 默认 `duration` 和 `delay` 需要**有单位**，无单位的写法可能会造成未定义的行为。

:::
