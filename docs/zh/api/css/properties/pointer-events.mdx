# pointer-events

## 介绍

CSS 属性 `pointer-events` 指定元件是否能够响应触摸事件。

## 语法

```css
pointer-events: auto;
pointer-events: none;
```

### 取值

- `auto`

  默认值，此时，节点可以响应触摸事件。

- `none`

  节点不能响应触摸事件。

## 形式定义

import { PropertyDefinition } from '@/components/PropertyDefinition';

## 形式语法

```
pointer-events: auto | none;
```

## 与 Web 的区别

不支持 `inherit`、`initial` 和 `unset`，不存在 `visiblePaintedroll`、 `visibleFill` 等 `SVG only` 的属性值。

## 注意事项

- 该属性有继承效果，即当子节点没有设置 `pointer-events` 时，会继承父节点的 `pointer-events` 值。
