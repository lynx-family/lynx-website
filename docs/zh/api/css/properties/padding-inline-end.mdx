# padding-inline-end

## 介绍

定义了元件的逻辑行末内边距，并根据元件的 [`direction`](./direction) 对应至实体内边距。该属性不能为负值。

在 `LTR` 模式下，`padding-inline-end` 等价于 [`padding-right`](./padding-right)。
在 `RTL` 模式下，`padding-inline-end` 等价于 [`padding-left`](./padding-left)。

## 语法

```css
/* <length> values */
padding-inline-end: 0.5em;
padding-inline-end: 0px;
padding-inline-end: 2rpx;

/* <percentage> value */
padding-inline-end: 10%;
```

### 取值

#### `0px`

**默认值**。

#### `<length>`

[`<length>`](/api/css/data-type/length.mdx) 定义为一个绝对值。不能为负值。

#### `<percentage>`

相对于[包含块](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Containing_block)的行内尺寸（宽度）的百分比。必须是非负值。

```

  <length-percentage [0,∞]>

<length-percentage> =
  <length>      |
  <percentage>
```
