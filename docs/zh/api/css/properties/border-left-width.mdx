# border-left-width

## 介绍

用于设置左边框的宽度。

## 语法

```css
border-left-width: thin;
border-left-width: medium;
```

### 取值

- `thin`
  细边框。

- `medium`
  中等边框。

- `thick`
  粗边框。

- [`<length>`](api/css/data-type/length.mdx)
  以具体的尺寸定义边框宽度

## 形式语法

```css
border-left-width = <line-width>
<line-width> = thin | medium | thick | <length>

```

## 与 Web 的区别

- [mdn:border-left-width](https://developer.mozilla.org/en-US/docs/Web/CSS/border-left-width)

- 不支持 `inherit、initial、revert、unset` 等

- 默认值为 0, 区别于 web 的默认值 'medium'
