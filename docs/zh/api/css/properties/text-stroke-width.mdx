## 介绍

`text-stroke-width` 属性指定了文字的笔画宽度。

## 使用示例

  example="css-api"

/>

## 语法

```css
text-stroke-width: 2px;
text-stroke-width: 0.1em;
text-stroke-width: 5pt;
```

### 取值

- [`<length>`](/api/css/data-type/length.mdx)<br/>
  设置笔画的宽度

## 与 Web 的区别

- 不支持设置笔画宽度为 thin 等属性值。
