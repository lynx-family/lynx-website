# -x-handle-size

## 介绍

`-x-handle-size` 属性指定了复制文本时浮标的大小，只在 selection 伪元件中有效。

## 语法

```css
::selection {
  -x-handle-size: 5px;
}
```

### 取值

- [`<length>`](api/css/data-type/length.mdx)<br/>
  设置浮标的大小

## 形式语法

```
-x-handle-size = <length>
```

## 与 Web 的区别

- Lynx 特有的属性。

## 注意事项

- Android 上对应整个浮标大小，iOS 上对应浮标圆球的直径
