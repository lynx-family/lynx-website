# relative-id

## 介绍

用于设置[相对布局](/guide/ui/layout/relative-layout.mdx)中同级元件的编号。`-1` 和 `0` 是特殊值。`-1` 是默认值，代表当前元件不会被视为同级元件的对齐参考标准。不能使用 `0` 作为元件的 id，否则有不符合预期的表现。

## 语法

```css
relative-id: 1;
relative-id: 2;
```

### 取值

- `<number>`

  `int` 值。表示当前元件的编号。**默认值**为 `-1`。`-1` 和 `0` 是特殊值。`-1` 是默认值，代表当前元件不会被视为同级元件的对齐参考标准。不能使用 `0` 作为元件的 id，否则有不符合预期的表现。
