# visibility

## 介绍

CSS 属性 visibility 显示或隐藏元件而不更改文档的布局。

## 语法

```css
/* Keyword values */
visibility: visible;
visibility: hidden;
```

### 取值

hidden,visible

- **visible**
  默认值，元件正常显示。

- hidden
  隐藏元件，但是其他元件的布局不改变，同时会隐藏其所有孩子，跟 `opacity:0` 效果类似。

## 形式语法

```
visibility =
  visible   |
  hidden
```

## 与 Web 的区别

不支持 `collapse`、`inherit`、`initial` 和 `unset`
如果设置为 hidden，但其子元件设为 visibility: visible，则该子元件依然不可见
