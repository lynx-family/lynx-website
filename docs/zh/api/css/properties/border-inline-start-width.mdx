# border-inline-start-width

## 介绍

用于设置阅读模式结束方向的边框宽度。

在 `LTR` 模式下，`border-inline-start-width` 等价于 `border-left-width`。
在 `RTL` 模式下，`border-inline-start-width` 等价于 `border-right-width`。

## 语法

```css
border-inline-start-width: thin;
border-inline-start-width: medium;
border-inline-start-width: thick;
border-inline-start-width: 10px;
```

### 取值

- `thin`
  细边框。

- `medium`
  中等边框。

- `thick`
  粗边框。

- [`<length>`](/api/css/data-type/length)
  以具体的尺寸定义边框宽度。

- 默认值
  **0px**
