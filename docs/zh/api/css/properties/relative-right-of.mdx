# relative-right-of

## 介绍

[相对布局](guide/ui/layout/relative-layout.mdx)中，设置当前元件在指定对应 [`id`](./relative-id.mdx) 的同级元件的右侧。

## 取值

### `none`

**默认值**。不设置此属性。

### `<number>`

`int` 值。设置当前元件在指定对应 [`id`](./relative-id.mdx) 的同级元件的右侧。

:::info
[`id`](./relative-id.mdx) 的设定请注意：`-1` 和 `0` 是特殊值。`-1` 是默认值，代表当前元件不会被视为同级元件的对齐参考标准。不能使用 `0` 作为元件的 id，否则有不符合预期的表现。
:::

## 语法

```css
relative-right-of: none;
relative-right-of: 1;
relative-right-of: 2;
```
