# font-feature-settings

## 介绍

`font-feature-settings` 属性用于控制 OpenType 字体中的高级印刷功能。

## 语法

```css
font-feature-settings: 'onum';
```

import { PropertyDefinition } from '@/components/PropertyDefinition';

## 形式语法

```
font-feature-settings =
  normal                |
  <feature-tag-value>#

<feature-tag-value> =
  <opentype-tag> [ <integer [0,∞]> | on | off ]?

<opentype-tag> =
  <string>
```
