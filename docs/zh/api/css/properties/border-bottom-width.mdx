# border-bottom-width

## 介绍

用于设置下边框的宽度。

## 语法

```css
border-bottom-width: 10px;
```

### 取值

- `thin`：细边框（2px）
- `medium`：中等边框（4px）
- `thick`：粗边框（6px）
- [`<length>`](/api/css/data-type/length.mdx)：具体尺寸值

## 形式语法

```css
border-bottom-width = <line-width>
<line-width> = thin | medium | thick | <length>

```

## 与 Web 的区别

- [MDN 参考](https://developer.mozilla.org/zh-CN/docs/Web/CSS/border-bottom-width)
- 具体尺寸值不同（Web 标准：`thin=1px, medium=3px, thick=5px`）
- 默认值不同（Web 为 `medium`）
- 不支持 `inherit/initial/revert/unset` 全局值
