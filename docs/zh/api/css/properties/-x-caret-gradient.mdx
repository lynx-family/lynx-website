# -x-caret-gradient

<APISummary />

## 介绍

`-x-caret-gradient` 属性设置可编辑元件中文本插入光标的渐变。

## 语法

```css
input {
  -x-caret-gradient: linear-gradient(to bottom, #3b82f6, #a855f7);
}
```

### 取值

- `none`<br/>
  使用默认光标绘制。

- [`<gradient>`](/api/css/data-type/gradient.mdx)<br/>
  设置光标的渐变画刷。
