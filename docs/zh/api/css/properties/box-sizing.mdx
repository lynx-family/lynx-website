# box-sizing

## 介绍

`box-sizing` 决定元件的 [`width`](./width.mdx)、[`height`](./height.mdx)、[`min-width`](./min-width.mdx)、[`min-height`](./min-height.mdx)、[`max-width`](./max-width.mdx)、[`max-height`](./max-height.mdx) 如何解析。

## 语法

```css
box-sizing: border-box;
box-sizing: content-box;
```

### 取值

- `border-box`

  **默认值。**[`width`](./width.mdx) 和 [`height`](./height.mdx) 属性包括内容 (`content`)，内边距（`padding`），和边框（`border`），但不包括外边距（`margin`）。

  尺寸计算公式：

  width = border + padding + content

  height = border + padding + content

- `content-box`

  标准盒子模型。[`width`](./width.mdx) 和 [`height`](./height.mdx) 只包括内容 (`content`)，不包括边框（`border`），内边距（`padding`），外边距（`margin`）。

  尺寸计算公式：

  width = content

  height = content

## 形式语法

```
box-sizing = border-box | content-box
```

## 与 Web 的区别

- Lynx 的默认值为 `border-box`，Web 的默认值为 `content-box`。
- `box-sizing` 不会影响 [`flex-basis`](./flex-basis.mdx) 的取值。
