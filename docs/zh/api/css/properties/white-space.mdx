# white-space

## 介绍

`white-space` 设置如何处理元件中的空白。

## 语法

```css
white-space: normal;

white-space: nowrap;
```

### 取值

- `normal`
  **默认值。**
  不对文本做额外处理。

- `nowrap`
  文本内的换行无效。

## 形式定义

<PropertyDefinition

## 形式语法

```
normal | nowrap
```

## 与 Web 的区别

- 只支持 `normal` 和 `nowrap` 两个关键字，和 Web 表现不同。
