# font-optical-sizing

## 介绍

`font-optical-sizing` 属性用于设置文本渲染是否针对不同大小进行显示优化。

## 语法

```css
font-optical-sizing: auto;
```

## 形式语法

```
font-optical-sizing =
  auto | none
```

## 与 Web 的区别

- 与 Web 默认值是 `auto` 不同，Lynx 中默认值为 `none`。 `font-optical-sizing` 属性是对 `font-variation-settings` 属性的补充，光学尺寸变化轴在 `font-variation-settings` 中由 `opsz` 表示。
