# border-right-width

## 介绍

用于设置右边框的宽度。

## 语法

```css
border-right-width: thin;
border-right-width: medium;
```

### 取值

- `thin`
  细边框。

- `medium`
  中等边框。

- `thick`
  粗边框。

- [`<length>`](/api/css/data-type/length.mdx)
  以具体的尺寸定义边框宽度

## 形式语法

```
<line-width>
where
<line-width> = <length> | thin | medium | thick

```

## 与 Web 的区别

- [mdn:border-right-width](https://developer.mozilla.org/en-US/docs/Web/CSS/border-right-width)

不支持 inherit、initial、revert、unset 等

默认值为 0, 区别于 web 的默认值 'medium'
