# transition-delay

## 介绍

用于设置过渡动画延迟执行的时间

## 语法

```css
transition-delay: 2s;
transition-delay: 300ms;
```

### 取值

- [`<time>`](api/css/data-type/time.mdx)
  延迟时间。
- 默认值
  **0s**
