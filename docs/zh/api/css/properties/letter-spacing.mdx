# letter-spacing

## 介绍

`letter-spacing` 属性用于设置文本字符的间距表现。

## 使用示例

## 语法

```css
/* <length> values */
letter-spacing: 0.3em;
letter-spacing: 3px;
letter-spacing: 0.3px;
```

### 取值

- [`<length>`](api/css/data-type/length.mdx)
  指定文字间的间距以替代默认间距。可以是负值。

```

```

## 与 Web 的区别

- 不支持 `normal` 关键字。只支持 `<length>`。
