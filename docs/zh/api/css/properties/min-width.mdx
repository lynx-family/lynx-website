# min-width

## 介绍

设置元件的最小宽度。它可以阻止 [`width`](./width.mdx) 属性的应用值小于 `min-width` 的值。
`min-width` 的值会覆盖 [`width`](./width.mdx)，但会被 [`max-width`](./max-width.mdx) 覆盖。

## 语法

```css
/* <length> */
min-width: 120px;
min-width: 10em;

/* <percentage> */
min-width: 75%;
```

### 取值

#### `0px`

**默认值**。

#### `<length>`

[`<length>`](api/css/data-type/length.mdx) 定义了一个确定值。

#### `<percentage>`

以包含区块的宽度百分比定义 `min-width`。

## 形式语法

```
min-width = <length-percentage [0,∞]>

<length-percentage> =
  <length>      |
  <percentage>
```

## 与 Web 的区别

[mdn: min-width](https://developer.mozilla.org/en-US/docs/Web/CSS/min-width)

1. 不支持 `auto`，默认值是 0px。
2. 不支持 `none`，`max-content`，`min-content`，`fit-content`。
3. 在 Lynx 中 `min-width` 不会覆盖 [`max-width`](./max-width.mdx)。
