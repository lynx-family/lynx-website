# border

## 介绍

用于设置所有边框相关样式，该属性可按顺序设置 border-width、border-style、border-color。

## 语法

```css
/* style */
border: solid;

/* width | style */
border: 2px dotted;

/* style | color */
border: outset #f33;

/* width | style | color */
border: medium dashed green;
```

### 取值

可按如下顺序设置：

- [`border-width`](./border-width.mdx)
  边框的宽度

- [`border-style`](./border-style.mdx)
  边框的样式

- [`border-color`](./border-color.mdx)
  边框的颜色

## 形式语法

```
<line-width> || <line-style> || <color>
where
<line-width> = <length> | thin | medium | thick
<line-style> = none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset

```

## 与 Web 的区别

- [mdn:border](https://developer.mozilla.org/en-US/docs/Web/CSS/border)

不支持 inherit、initial、revert、unset 等
