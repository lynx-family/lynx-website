# outline-color

## 介绍

用于设置轮廓的颜色。

## 使用示例

![demo](https://lf-lynx.tiktok-cdns.com/obj/lynx-artifacts-oss-sg/lynx-website/assets/doc/IMG_3701.jpeg)

### 取值

- [`<color>`](api/css/data-type/color.mdx)
  指定轮廓颜色

## 形式语法

```
<color>
```

样例：

```css
/* 轮廓是红色 */
outline-color: red;

/* 轮廓是绿色 */
outline-color: rgb(0, 255, 0);

/* 轮廓是蓝色 */
outline-color: #0000ff;
```

## 与 Web 的区别

- [mdn:outline-color](https://developer.mozilla.org/en-US/docs/Web/CSS/outline-color)

不支持 inherit、initial、revert、unset、invert 等
