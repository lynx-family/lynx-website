# -x-handle-color

## 介绍

`-x-handle-color` 属性指定了复制文本时浮标的颜色，只在 selection 伪元件中有效。

## 使用示例

## 语法

```css
::selection {
  -x-handle-color: blue;
}
```

### 取值

- [`<color>`](./color.mdx)<br/>
  设置浮标的颜色
