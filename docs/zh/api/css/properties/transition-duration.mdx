# transition-duration

## 介绍

用于设置过渡动画执行的时间

## 语法

```css
transition-duration: 2s;
transition-duration: 300ms;
```

### 取值

- [`<time>`](/api/css/data-type/time.mdx)
  执行时间。

- 默认值
  **0s**
