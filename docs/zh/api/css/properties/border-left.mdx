# border-left

## 介绍

设置元件左边框样式的简写属性，按顺序包含 [`border-left-width`](./border-left-width)、[`border-left-style`](./border-left-style)、[`border-left-color`](./border-left-color)。

## 语法

```css
border-left: 1px solid red;
```

## 取值

可按顺序设置如下属性：

- [`border-left-width`](./border-left-width.mdx)
  边框的宽度

- [`border-left-style`](./border-left-style.mdx)
  边框的样式

- [`border-left-color`](./border-left-color.mdx)
  边框的颜色

```css
border-left = <line-width> <line-style> <color>
```

## 与 Web 的区别

- [mdn:border-left](https://developer.mozilla.org/en-US/docs/Web/CSS/border-left)

- 不支持 `inherit、initial、revert、unset` 等
