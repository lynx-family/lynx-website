# -x-caret-height

<APISummary />

## 介绍

`-x-caret-height` 属性设置可编辑元件中文本插入光标的高度。

## 语法

```css
input {
  -x-caret-height: 20px;
}
```

### 取值

- [`<length>`](/api/css/data-type/length.mdx)<br/>
  设置光标的高度。
