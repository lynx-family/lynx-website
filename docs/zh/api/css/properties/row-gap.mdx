# row-gap (grid-row-gap)

## 介绍

`row-gap`/`grid-row-gap` 属性用来设置[弹性布局](guide/ui/layout/flexible-box-layout.mdx)和[网格布局](guide/ui/layout/grid-layout.mdx)行元件之间的间隙（gutter）大小。

为了兼容性，可使用 `row-gap` 或 `grid-row-gap`。

## 取值

### `<length>`

[`<length>`](api/css/data-type/length.mdx) 表示[弹性布局](guide/ui/layout/flexible-box-layout.mdx)或[网格布局](guide/ui/layout/grid-layout.mdx)行之间的间隙宽度。

### `<percentage>`

行之间直接的间隙宽度，[`<percentage>`](api/css/data-type/percentage.mdx) 相对容器自身 content area **纵向**大小的百分比。

## 形式语法

## 语法

```css
/* <length> values */
row-gap: 20px;
grid-row-gap: 20px;
row-gap: 1em;
grid-row-gap: 1em;

/* <percentage> value */
row-gap: 10%;
grid-row-gap: 10%;

/* calc() values */
row-gap: calc(10% + 20px);
grid-row-gap: calc(10% + 20px);
```
