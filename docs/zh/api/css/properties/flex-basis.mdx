# flex-basis

## 介绍

`flex-basis` 指定了弹性项目在**主轴**方向上的初始大小。这个属性就决定了弹性项目的边框区域（border area）的尺寸。
当一个元件同时被设置了 `flex-basis` (除值为 `auto` 外) 和 [`width`](./width.mdx) (或者在 [`flex-direction: column`](./flex-direction.mdx) 情况下设置了 [`height`](./height.mdx))，`flex-basis` 具有更高的优先级。

## 语法

```css
/* Specify <'width'> */
flex-basis: 10em;
flex-basis: 3px;
flex-basis: 50%;
flex-basis: auto;
```

### 取值

- `auto`

  **默认值**。参照自身的 [`width`](./width.mdx) 和 [`height`](./height.mdx) 属性。

- `<length>`

  绝对尺寸大小。

- `<percentage>`

  一个相对于其父弹性容器**主轴**内容区域（content area）的百分数。

## 形式语法

```
flex-basis = auto | <length-percentage [0,∞]>

<length-percentage> =
  <length>      |
  <percentage>
```

## 与 Web 的区别

- 值不受 [`box-sizing`](./box-sizing.mdx) 影响。
