:::warning
该 API 已被废弃。
:::

## 介绍

在[网格布局](guide/ui/layout/grid-layout.mdx)中，指定网格项目的行跨度。用于简化开发，W3C 没有这个属性。此属性是子元件属性。

当网格项目指定了 [`grid-row-start`](./grid-row-start.mdx) 和 [`grid-row-end`](./grid-row-end.mdx) 时，`grid-row-span` 属性失效。

当在 `grid-row-start` 或 `grid-row-end` 中使用带有 span 关键字的值时，不能同时使用 `grid-row-span` 属性，否则会出现不符合预期的行为。

## 语法

```css
/* <number> values */
grid-row-span: 1;
grid-row-span: 2;
```

### 取值

- `<number>`

  [`<number>`](api/css/data-type/number.mdx) 整数数字，表示网格项目在行方向上的跨度。暂不支持负数。
