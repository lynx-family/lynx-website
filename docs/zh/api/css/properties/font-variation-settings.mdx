# font-variation-settings

## 介绍

`font-variation-settings` 属性为可变字体特性提供了底层控制，让你可以指定要改变的特性的四字母轴名称及其值。

## 语法

```css
font-variation-settings: 'wght' 750;
```

## 形式语法

```
font-variation-settings =
  normal                        |
  [ <opentype-tag> <number> ]#

<opentype-tag> =
  <string>
```
