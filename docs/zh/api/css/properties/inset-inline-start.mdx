# inset-inline-start

## 介绍

CSS 属性 `inset-inline-start` 定义了元件的逻辑行首偏移，并根据元件的行内方向对应至实体偏移。根据 [`direction`](./direction) 所定义的值，此属性对应于 [`right`](./right) 或 [`left`](./left) 属性。

在 `LTR` 模式下，`inset-inline-start` 等价于 `left`。
在 `RTL` 模式下，`inset-inline-start` 等价于 `right`。

## 语法

```css
/* <length> values */
inset-inline-start: 3px;
inset-inline-start: 2rpx;
inset-inline-start: 2.4em;
inset-inline-start: 3rem;

/* 参照于包含块宽度的百分比 */
inset-inline-start: 10%;

/* Keyword value */
inset-inline-start: auto;

/* calc */
inset-inline-start: calc(1px + 1px);
```

### 取值

`inset-inline-start` 属性的取值与 [`left`](./left) 属性相同。
