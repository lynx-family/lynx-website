# -x-auto-font-size

## 介绍

`-x-auto-font-size` 属性设置是否开启文本自适应调整字号大小功能，最小的文本字号，最大的文本字号，字号调整的粒度值，除一个参数其余可省略。

## 语法

```css
-x-auto-font-size: 'true';
-x-auto-font-size: 'true 10px';
-x-auto-font-size: 'true 10px 30px';
-x-auto-font-size: 'true 10px 30px 2px';
```

### 取值

- 第一个参数，表示是否开启文本自适应调整字号大小功能，true 表示开启，false 表示关闭

- 第二个参数，表示最小的文本字号，取值为 [`<length>`](api/css/data-type/length.mdx)，此参数可省略

- 第三个参数，表示最大的文本字号，取值为 [`<length>`](api/css/data-type/length.mdx)，此参数在存在前两个参数时可省略

- 第四个参数，表示字号调整的粒度值，取值为 [`<length>`](api/css/data-type/length.mdx)，此参数在存在前三个参数时可省略，默认值为 1px
