# align-self

import { AvailableBadge } from '@lynx';

## 介绍

[弹性布局](guide/ui/layout/flexible-box-layout.mdx)、[线性布局](guide/ui/layout/linear-layout.mdx)中，`align-self` 会在**交叉轴上**对齐当前容器中的元件。[网格布局](guide/ui/layout/grid-layout.mdx)中，`align-self` 控制子元件在其网格区域（grid area）内的**块向轴**上的对齐，且 `align-self` 会覆盖已有的 [`align-items`](./align-items.mdx) 的值。

对于[弹性布局](guide/ui/layout/flexible-box-layout.mdx)，如果任何子元件的交叉轴方向 [`margin`](./margin.mdx) 值设置为 auto，则会忽略 `align-self`。

:::info
对于[网格布局](guide/ui/layout/grid-layout.mdx)，Lynx 不支持 `writing-mode`，所以块轴就是垂直轴。
:::

## 语法

```css
/* 基本关键字 */
align-self: stretch;
align-self: auto;

/* 定位对齐 */
align-self: center;
align-self: start;
align-self: end;
align-self: flex-start;
align-self: flex-end;

/* 基线对齐 */
align-self: baseline;
```

### 取值

#### `auto`

**默认值**。根据 [`align-items`](./align-items.mdx) 定义的方式对齐。

#### `stretch`

元件在**交叉轴**（[网格布局](guide/ui/layout/grid-layout.mdx)是块轴）方向上未设置尺寸（高度/宽度）或设为 `auto`，会被拉伸到与行相同的高度或列相同的宽度。同时这些元件仍然保持其宽高比例的约束。

#### `center`

在[弹性布局](guide/ui/layout/flexible-box-layout.mdx)、[线性布局](guide/ui/layout/linear-layout.mdx)中，元件在交叉轴居中。

在[网格布局](guide/ui/layout/grid-layout.mdx)中，元件在网格区域的块轴方向上居中。

#### `start`

在[弹性布局](guide/ui/layout/flexible-box-layout.mdx)、[线性布局](guide/ui/layout/linear-layout.mdx)中，元件在交叉轴上对齐起始位置。

在[网格布局](guide/ui/layout/grid-layout.mdx)中，元件在块轴上与网格区域的起始边缘对齐。

#### `end`

在[弹性布局](guide/ui/layout/flexible-box-layout.mdx)、[线性布局](guide/ui/layout/linear-layout.mdx)中，元件在交叉轴上对齐结尾位置。

在[网格布局](guide/ui/layout/grid-layout.mdx)中，元件在块轴上与网格区域的结束边缘对齐。

#### `flex-start`

同 `start` 表现一致。

#### `flex-end`

同 `end` 表现一致。

#### `baseline`

元件参与基线对齐。目前仅支持在[弹性布局](guide/ui/layout/flexible-box-layout.mdx)中使用。

## 形式语法

```
align-self = auto | center | start | end | flex-start | flex-end | baseline | stretch
```

## 与 Web 的区别

- 不支持 `normal`、`self-start` 和 `self-end`。
