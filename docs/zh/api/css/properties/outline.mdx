# outline

## 介绍

用于设置轮廓的样式。该属性可按顺序设置 outline-width、outline-style、outline-color。

`border` 和 `outline` 很类似，但有如下区别：

- 轮廓不占据空间，绘制于元件内容周围。
- 根据规范，轮廓通常是矩形，但也可以是非矩形的。

### 取值

可按顺序设置如下属性：

- [`outline-width`](./outline-width.mdx)
  轮廓的宽度

- [`outline-style`](./outline-style.mdx)
  轮廓的样式

- [`outline-color`](./outline-color.mdx)
  轮廓的颜色

## 形式语法

```
[ <outline-color> || <outline-style> || <outline-width> ]
```

样例：

```css
/* 样式 */
outline: solid;

/* 颜色 | 样式 */
outline: #f66 dashed;

/* 样式 | 宽度 */
outline: inset thick;

/* 颜色 | 样式 | 宽度 */
outline: green solid 3px;
```

## 与 Web 的区别

- [mdn:outline](https://developer.mozilla.org/en-US/docs/Web/CSS/outline)

不支持 inherit、initial、revert、unset 等
