# PerformanceObserver

`PerformanceObserver` 用于观察性能事件，并在 [Lynx Engine](/guide/spec#engine) 产生性能事件 ([`PerformanceEntry`](./performance-entry)) 时收到通知。
它必须通过 [`lynx.performance.createObserver()`](/api/lynx-api/lynx/lynx-performance#createobserver) 创建，并在创建时绑定一个回调函数用于处理接收到的性能事件。

## 实例方法

[`PerformanceObserver.observe()`](./performance-observer/observe)

指定要观察的性能事件类型集合并开始观察。

[`PerformanceObserver.disconnect()`](./performance-observer/disconnect)

停止接收性能事件。
