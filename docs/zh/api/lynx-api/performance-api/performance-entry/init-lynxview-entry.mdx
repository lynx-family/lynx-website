# InitLynxviewEntry

`InitLynxviewEntry` 接口提供有关 [LynxView](guide/spec#lynxview) 初始化的关键时间，它继承自 [`PerformanceEntry`](/api/lynx-api/performance-api/performance-entry)。

## 实例属性

### [entryType](/api/lynx-api/performance-api/performance-entry#entryType)

```tsx
entryType: string;
```

性能事件的类型，当前类所有实例的值均固定为 `init`。

### [name](/api/lynx-api/performance-api/performance-entry#name)

```tsx
name: string;
```

性能事件的名称，当前类所有实例的值均固定为 `lynxview`。

### createLynxStart

```tsx
createLynxStart: number;
```

LynxView 开始创建的时间。该时间戳是一个表示为浮点型的 Unix 时间戳（单位：毫秒），精确到小数点后三位。例如：1739594612307.429。

### createLynxEnd

```tsx
createLynxEnd: number;
```

LynxView 创建完毕的时间。该时间戳是一个表示为浮点型的 Unix 时间戳（单位：毫秒），精确到小数点后三位。例如：1739594612307.429。
