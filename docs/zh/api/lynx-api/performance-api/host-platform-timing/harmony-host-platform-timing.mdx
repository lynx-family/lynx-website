# HarmonyHostPlatformTiming

该接口描述了 [Lynx Pipeline](/guide/spec#lynx-pipeline) 中 Harmony 平台特有的关键阶段的性能数据。

## 实例属性

### hostPlatformType

```tsx
hostPlatformType: string;
```

描述 Host Platform 类型的字符串，默认取值为：`Harmony`。
