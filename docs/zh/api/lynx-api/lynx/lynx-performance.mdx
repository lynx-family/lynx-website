# lynx: performance readonly object

通过该对象访问当前页面与性能指标有关的信息。

## 实例方法

[`createObserver()`](./lynx-performance/create-observer) 创建一个 [`PerformanceObserver`](/api/lynx-api/performance-api/performance-observer) 对象，该对象将用于接收 Lynx 引擎产生的性能事件。

[`profileStart()`](./lynx-performance/profile-start) 标记一个 [trace event](https://perfetto.dev/docs/instrumentation/track-events) 的开始。

[`profileEnd()`](./lynx-performance/profile-end) 标记最近的 [trace event](https://perfetto.dev/docs/instrumentation/track-events) 的结束。

[`profileMark()`](./lynx-performance/profile-mark) 标记一个即时(没有持续时间) trace event。

[`profileFlowId()`](./lynx-performance/profile-flow-id) 为 [flow events](https://perfetto.dev/docs/instrumentation/track-events#flows) 生成一个唯一的 flow id.

[`isProfileRecording()`](./lynx-performance/is-profile-recording) 检查当前是否正在进行 trace 录制。
