# lynx: accessibilityAnnounce() static method

让屏幕阅读器朗读指定的文本内容，帮助视觉障碍用户感知界面状态变化或操作结果。

## 语法

```ts
accessibilityAnnounce({"content" : string}) : void;
```

### 参数

#### `content`

待朗读文本内容

## 示例

```jsx
lynx.accessibilityAnnounce(
  {
    content: 'hello lynx',
  },
  (res) => {
    console.log('test: ' + JSON.stringify(res));
  },
);
```
