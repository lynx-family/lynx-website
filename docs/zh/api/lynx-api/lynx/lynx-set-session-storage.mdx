# lynx: setSessionStorageItem() static method

<APISummary />

`lynx.setSessionStorageItem` 方法可以被用来设置多 LynxView 共享数据，即多页面之间通过 `SessionStorage` 实现数据共享，可以调用 [lynx.getSessionStorage](./lynx-get-session-storage.mdx) 来获取缓存的数据。

## 语法

```ts
setSessionStorageItem(key: string, value: object): void;
```

### 参数

#### key

设置缓存数据的 key，后续可通过该 key 来获取缓存数据以及监听数据变化。

#### value

需要存储的数据对象。

### 返回值

无（`undefined`）。
