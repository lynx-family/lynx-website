# lynx: queueMicrotask() static method

将一个任务发送到微任务队列执行。更多关于微任务的介绍，可参考[这里](https://developer.mozilla.org/en-US/docs/Web/API/Window/queueMicrotask)

## 语法

```ts
queueMicrotask(callback: () => void): void;
```

### 参数

#### callback

需要被发送到微任务队列的方法对象。

### 返回值

无（`undefined`）。

## 示例

```tsx
lynx.queueMicrotask(() => {
  console.log('microtask execute.');
});
```
