---
title: isProfileRecording()
---

检查当前是否正在进行 trace 录制。

## 语法

```typescript
isProfileRecording(): boolean
```

## 返回值

如果正在录制 trace，返回 `true`。

## 示例

```typescript
if (lynx.performance.isProfileRecording()) {
  lynx.performance.profileMark('RecordingIsActive');
}
```
