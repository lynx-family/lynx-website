# lynx: unsubscribeSessionStorage() static method

<APISummary />

`lynx.unsubscribeSessionStorage` 方法可以被用来取消监听多 LynxView 共享数据的变化，即多页面之间通过 `SessionStorage` 实现数据共享，可以调用 [lynx.setSessionStorageItem](./lynx-set-session-storage.mdx) 来设置缓存的数据。 取消监听后，该 key 对应的缓存数据将不再被监听。

## 语法

```ts
lynx.unsubscribeSessionStorage(key, id);
```

### 参数

#### key

缓存数据 key，通过该 key 来获取缓存数据以及监听数据变化。

#### id

监听数据变化的 id，通过 [lynx.subscribeSessionStorage](./lynx-subscribe-session-storage.mdx) 方法返回的 id 来取消监听。

### 返回值

无（`undefined`）。
