# SelectorQuery: exec() method

执行所有已提交的操作。注意不调用 `exec()` 则节点查找和节点操作都不会实际执行。

## 语法

```ts
exec(): void;
```

### 参数

无。

### 返回值

无（`undefined`）。

## 示例

```ts
lynx
  .createSelectorQuery()
  .select('#my-id')
  .invoke({
    method: 'boundingClientRect',
    success: function (res) {
      console.log(res);
    },
    fail: function (res) {
      console.log(res.code, res.data);
    },
  })
  .exec();
```
