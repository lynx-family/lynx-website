# console: countReset() method

`countReset` 方法用于重置 count() 方法的计数。

## 语法

```ts
countReset(label: string?): undefined;
```

### 参数

#### label

字符串。如果提供，则重置指定标签的计数，否则重置 "default" 标签的计数。

### 返回值

无（`undefined`）。
