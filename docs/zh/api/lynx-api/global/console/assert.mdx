# console: assert() method

`assert` 如果断言为 false，将错误消息写在控制台上。如果断言是真的，那么什么也不会发生。

## 语法

```ts
assert(assertion: boolean, val1, val2, ...): undefined;
```

### 参数

#### assertion

任何布尔表达。如果断言是错误的，则断言失败并将后续信息写入控制台。

#### val1, val2, ...

输出的 JavaScript 值列表。这些值中的每个值的表示形式都以给定的顺序输出。

### 返回值

无（`undefined`）。
