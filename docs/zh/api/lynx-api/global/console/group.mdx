# console: group() method

`group` 方法用于在控制台日志中创建一个新的分组，从而导致任何后续的控制台消息被附加级别缩进，直到调用 console.groupEnd() 。

## 语法

```ts
group(label: string?): undefined;
```

### 参数

#### label

group 的名称。

### 返回值

无（`undefined`）。
