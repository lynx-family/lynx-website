# console: count() method

`count` 方法用于记录该特定调用 count() 的次数。

## 语法

```ts
count(label: string?): undefined;
```

### 参数

#### label

字符串。如果提供，count() 输出该标签已调用的次数。如果省略，count() 的行为就像是用 "default" 标签调用的。

### 返回值

无（`undefined`）。
