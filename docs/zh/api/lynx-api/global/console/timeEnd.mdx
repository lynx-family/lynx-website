# console: timeEnd() method

`timeEnd` 方法用于停止以前通过调用 console.Time() 启动的计时器。

## 语法

```ts
timeEnd(label: string?): undefined;
```

### 参数

#### label

timer 的名称。

### 返回值

无（`undefined`）。
