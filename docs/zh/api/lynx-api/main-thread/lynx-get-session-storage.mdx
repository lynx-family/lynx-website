# lynx: getSessionStorageItem() static method

<APISummary />

`lynx.getSessionStorageItem` 方法可以被用来获取多 LynxView 共享数据，即多页面之间通过 `SessionStorage` 实现数据共享，可以调用 [lynx.setSessionStorageItem](./lynx-set-session-storage.mdx) 来设置缓存的数据。

## 语法

```ts
let value = lynx.getSessionStorageItem(key);
```

### 参数

#### key

缓存数据 key，通过该 key 来获取缓存数据以及监听数据变化。

### 返回值

缓存的数据对象。
