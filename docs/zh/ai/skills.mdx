# Lynx Agent Skills

[Agent Skills](https://github.com/anthropics/skills) 是可复用的规则集，能够为 coding agent 提供特定领域的知识。它们在编辑器中直接提供自动检测、最佳实践指导和自动修复能力。

Lynx 社区在 [lynx-community/skills](https://github.com/lynx-community/skills) 维护了一组 skills，帮助 coding agent 理解 Lynx 独特的架构和模式。

## 可用 Skills

### reactlynx-best-practices

教会 agent Lynx 的双线程架构模式。包含静态分析规则和常见问题的自动修复：

- **`detect-background-only`**（关键）— 确保原生 API 调用（`lynx.getJSModule`、`NativeModules`）只在后台线程上下文中发生，防止 UI 阻塞。
- **`proper-event-handlers`** — ReactLynx 中正确的事件处理模式（`bindtap`、`catchtap`）和事件传播。
- **`main-thread-scripts-guide`** — 如何正确使用 `'main thread'` 指令和 `main-thread:` 事件前缀来实现流畅的动画和手势处理。
- **`hoist-static-jsx`** — 通过将静态 JSX 元素提升到组件外部以避免不必要的重新创建来优化性能。

:::tip
这些规则也作为独立指南进行了文档化。详细说明和代码示例请参阅[最佳实践](/react/best-practices)。
:::

### devtool

基于 CLI 的 skill，让 agent 与 Lynx DevTool 交互，在已连接的设备（Android、iOS、桌面端）上检查和调试运行中的 Lynx 应用。该 skill 提供 Node.js CLI（`node scripts/index.mjs`），agent 可直接调用，无需 MCP 服务器。

:::info DevTool Skill 与 DevTool MCP
coding agent 与 Lynx DevTool 交互有两种方式：

- **DevTool Skill**（本页）— 基于 CLI，适用于任何 coding agent（Claude Code、Cursor、Copilot），通过 `npx skills add` 安装。
- **[DevTool MCP](/ai/lynx-devtool-mcp)** — MCP 服务器协议，适用于支持 MCP 的客户端。

两者提供类似的能力，根据你的 agent 配置选择合适的方式。
:::

**DOM 检查** — 使用 `querySelector` 查询元素，通过 `getDocument` 获取完整文档树，读取任意节点的 `getOuterHTML` 和 `innerText`。

**CSS 检查** — 读取任意元素的 `getComputedStyle`、`getMatchedStyles` 和 `getInlineStyles`，了解应用的样式。

**控制台日志** — 读取控制台输出，支持按级别（log、warn、error）和时间范围过滤，聚焦特定时间窗口。

**截图捕获** — 截取当前 Lynx 页面的屏幕截图，用于视觉验证和多模态分析。

**触摸模拟** — 通过 `emulateTouchFromMouseEvent` 模拟触摸事件，用于自动化交互测试（点击、拖动、滚动）。

**页面导航** — 在开发流程中重新加载页面或重定向到本地开发服务器 URL。在已连接的 Lynx 应用中打开任意 URL。

**JavaScript 执行** — 在 Lynx 运行时中直接执行 JavaScript 表达式，用于检查状态或触发操作。

**源码列表** — 列出 Lynx 运行时加载的所有已解析脚本，便于将控制台堆栈跟踪映射回源文件。

**故障排查** — 内置连接器和传输问题的调试日志。启用调试命名空间来诊断 agent 与 DevTool 之间的连接问题。

### lynx-error-remap

帮助 agent 调试主线程运行时错误，将压缩后的错误信息重新映射回原始源码，使堆栈跟踪变得可操作。

### lynx-ptrace

使 agent 能够分析 Lynx 性能追踪，识别模板加载、JS App 加载、跨线程调用和渲染管线中的瓶颈。

## 安装

使用一条命令安装 skills 集合：

```bash
npx skills add lynx-community/skills
```

这会将 skill 规则添加到你的项目中，以便兼容的 coding agent（如 Claude Code、Cursor 或 Copilot）能够自动应用它们。
