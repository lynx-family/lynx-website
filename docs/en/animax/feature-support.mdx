# Feature Support

This document lists AnimaX support for common After Effects features and compares those capabilities with Lottie Android and Lottie iOS.

The AnimaX version value indicates the first AnimaX version that supports the corresponding feature. For example, `1.0+` means the feature is supported starting from AnimaX 1.0. For release details and future version updates, see [AnimaX releases](https://github.com/lynx-family/animax/releases).

Lottie feature support data is based on [LottieFiles Supported Features](https://lottiefiles.com/supported-features).
