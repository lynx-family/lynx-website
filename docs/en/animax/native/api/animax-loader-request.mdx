# AnimaXLoaderRequest

`AnimaXLoaderRequest` describes the resource to load. It is usually passed to [AnimaXLoaderProtocol](./animax-loader-protocol).

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

## Creation

### requestWithURL

Creates a load request from a URL.

**Parameters**

| Name  | Type         | Description   |
| ----- | ------------ | ------------- |
| `url` | `NSString *` | Resource URL. |

**Return value**

`AnimaXLoaderRequest *`

### requestWithURL:requestProperties

Creates a load request from a URL and additional properties.

**Parameters**

| Name         | Type             | Description                        |
| ------------ | ---------------- | ---------------------------------- |
| `url`        | `NSString *`     | Resource URL.                      |
| `properties` | `NSDictionary *` | Additional properties, may be nil. |

**Return value**

`AnimaXLoaderRequest *`

## Properties

| Name                | Type             | Description                        |
| ------------------- | ---------------- | ---------------------------------- |
| `url`               | `NSString *`     | Resource URL.                      |
| `requestProperties` | `NSDictionary *` | Additional properties, may be nil. |

</PlatformTabs.Tab>
<PlatformTabs.Tab platform="android">

## Request Info

### getUri

Returns the resource URI.

**Return value**

`String`

### getParams

Returns additional resource loading parameters.

**Return value**

`Map<String, Object>`

### getImageInfo

Returns image size information. Non-image requests may return `null`.

**Return value**

`IImageInfo`

## IImageInfo

### getWidth

Returns the image width.

**Return value**

`int`

### getHeight

Returns the image height.

**Return value**

`int`
