# AnimaXComposition

`AnimaXComposition` is a parsed animation composition. Create it asynchronously from JSON or a URI through [AnimaXCompositionFactory](./animax-composition-factory), then pass it to [AnimaXView](./animax-view) through `setComposition` to preload and reuse parsed animation data.

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

## Frame Info

### startFrame

Returns the start frame of the composition.

**Return value**

`CGFloat`

### endFrame

Returns the end frame of the composition.

**Return value**

`CGFloat`

</PlatformTabs.Tab>
<PlatformTabs.Tab platform="android">

## Frame Info

### getStartFrame

Returns the start frame of the composition. Returns `0` after the native object has been released.

**Return value**

`long`

### getEndFrame

Returns the end frame of the composition. Returns `0` after the native object has been released.

**Return value**

`long`

## Native Resource

### getNativePtr

Returns the native pointer held by the composition. Returns `0` after the native object has been released.

**Return value**

`long`

### release

Immediately releases the native resource held by this composition. Do not reuse the `AnimaXComposition` after release.
