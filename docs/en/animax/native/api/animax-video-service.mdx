# AnimaXVideoService

`AnimaXVideoService` maps to Android `IAnimaXVideoService` and lets Android integrations provide a custom video player. AnimaX asks this service for an `IVideoPlayer` when a custom player is needed.

## Create Player

### createVideoPlayer

Creates a custom video player instance. Return `null` to use the default player.

**Parameters**

| Name           | Type                | Description           |
| -------------- | ------------------- | --------------------- |
| `nativePlayer` | `long`              | Native player handle. |
| `config`       | `VideoPlayerConfig` | Video player config.  |

**Return value**

`IVideoPlayer` or `null`.

## Registration

```java
AnimaXServiceCenter.inst().registerService(
    IAnimaXVideoService.class,
    myVideoService);
```
