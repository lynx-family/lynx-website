# AnimaXCompositionListener

`AnimaXCompositionListener` receives parsing results from [AnimaXCompositionFactory](./animax-composition-factory) on Android. Successful parsing returns [AnimaXComposition](./animax-composition); failed parsing returns an error message.

## Parsing Result

### onCompositionReady

Called when animation composition parsing completes.

**Parameters**

| Name          | Type                                        | Description                   |
| ------------- | ------------------------------------------- | ----------------------------- |
| `composition` | [`AnimaXComposition`](./animax-composition) | Parsed animation composition. |

### onCompositionFailed

Called when animation composition parsing fails.

**Parameters**

| Name           | Type     | Description        |
| -------------- | -------- | ------------------ |
| `errorMessage` | `String` | Error information. |
