# AnimaXLoaderResponse

`AnimaXLoaderResponse` contains resource data or error information. A loader passes it back to AnimaX through a completion handler.

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

## Creation

### responseWithPayload

Creates a successful response. `payload` can be a file path, byte data, image, or another loaded resource.

**Parameters**

| Name      | Type            | Description               |
| --------- | --------------- | ------------------------- |
| `payload` | generic payload | Resource data on success. |

**Return value**

`AnimaXLoaderResponse *`

### responseWithError

Creates an error response from `NSError`.

**Parameters**

| Name    | Type        | Description        |
| ------- | ----------- | ------------------ |
| `error` | `NSError *` | Error information. |

**Return value**

`AnimaXLoaderResponse *`

### responseWithErrorMessage

Creates an error response from a message.

**Parameters**

| Name           | Type         | Description        |
| -------------- | ------------ | ------------------ |
| `errorMessage` | `NSString *` | Error information. |

**Return value**

`AnimaXLoaderResponse *`

## Properties

| Name      | Type            | Description                   |
| --------- | --------------- | ----------------------------- |
| `payload` | generic payload | Resource data on success.     |
| `error`   | `NSError *`     | Error information on failure. |

## Creation

### createStringFilePathResponse

Creates a successful response from a local file path.

**Parameters**

| Name       | Type     | Description      |
| ---------- | -------- | ---------------- |
| `filePath` | `String` | Local file path. |

**Return value**

`AnimaXLoaderResponse<String>`

### createByteArrayResponse

Creates a successful response from bytes.

**Parameters**

| Name    | Type     | Description    |
| ------- | -------- | -------------- |
| `bytes` | `byte[]` | Resource data. |

**Return value**

`AnimaXLoaderResponse<byte[]>`

### createBitmapResponse

Creates a successful response from a bitmap reference.

**Parameters**

| Name     | Type                              | Description       |
| -------- | --------------------------------- | ----------------- |
| `bitmap` | `IAnimaXCloseableBitmapReference` | Bitmap reference. |

**Return value**

`AnimaXLoaderResponse<IAnimaXCloseableBitmapReference>`

### createErrorResponse

Creates an error response from `Throwable`.

**Parameters**

| Name    | Type        | Description        |
| ------- | ----------- | ------------------ |
| `error` | `Throwable` | Error information. |

**Return value**

`AnimaXLoaderResponse<Throwable>`

## Response Data

### getData

Returns the data carried by this response.

**Return value**

`T`
