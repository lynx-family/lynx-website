# AnimaXLoaderProtocol

`AnimaXLoaderProtocol` provides custom resource loading. It is usually created and returned by [AnimaXResourceFactoryService](./animax-resource-factory-service).

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

## Load Request

### handleRequest

Executes the load request. When loading ends, pass [AnimaXLoaderResponse](./animax-loader-response) back to AnimaX through [AnimaXLoaderCompletionHandler](./animax-loader-completion-handler).

**Parameters**

| Name         | Type                                                                  | Description          |
| ------------ | --------------------------------------------------------------------- | -------------------- |
| `request`    | [`AnimaXLoaderRequest`](./animax-loader-request) `*`                  | Load request.        |
| `completion` | [`AnimaXLoaderCompletionHandler`](./animax-loader-completion-handler) | Completion callback. |

## Resource Source

### getScheme

Returns the resource source handled by this loader.

**Return value**

`getScheme` returns [AnimaXLoaderScheme](./animax-loader-scheme).

## Load Request

### load

Executes the load request. When loading ends, pass [AnimaXLoaderResponse](./animax-loader-response) back to AnimaX through [AnimaXLoaderCompletionHandler](./animax-loader-completion-handler).

**Parameters**

| Name                | Type                                                                  | Description          |
| ------------------- | --------------------------------------------------------------------- | -------------------- |
| `request`           | [`AnimaXLoaderRequest`](./animax-loader-request)                      | Load request.        |
| `completionHandler` | [`AnimaXLoaderCompletionHandler`](./animax-loader-completion-handler) | Completion callback. |

## Resource Source

### getScheme

Returns the resource source handled by this loader.

**Return value**

`getScheme` returns [AnimaXLoaderScheme](./animax-loader-scheme).
