# AnimaXResourceFactoryService

`AnimaXResourceFactoryService` creates custom resource loaders. AnimaX uses it to obtain loaders for resource requests.

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

Implement this protocol to provide custom resource loaders such as network, file, or asset loaders.

## Create Loaders

### createAnimaXLoaders

Creates and returns custom loaders. Each loader uses [AnimaXLoaderProtocol](./animax-loader-protocol) to declare the resource source it can handle.

**Return value**

`NSArray<id<AnimaXLoaderProtocol>> *`

## Registration

```objc
[[AnimaXServiceCenter sharedInstance] registerService:@protocol(AnimaXResourceFactoryService)
                                             withImpl:myFactory];
```

## Create Loaders

### createAnimaXLoaders

Creates and returns custom loaders. Each loader uses [AnimaXLoaderProtocol](./animax-loader-protocol) to declare the resource source it can handle.

**Return value**

`List<IAnimaXLoader>`

## Registration

```java
AnimaXServiceCenter.inst().registerService(
    IAnimaXResourceFactoryService.class,
    myFactory);
```
