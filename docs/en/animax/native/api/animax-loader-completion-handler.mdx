# AnimaXLoaderCompletionHandler

`AnimaXLoaderCompletionHandler` is the resource loading completion callback. Call it when loading ends and pass [AnimaXLoaderResponse](./animax-loader-response) back to AnimaX.

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

## Block Signature

```objc
typedef void (^AnimaXLoaderCompletionHandler)(AnimaXLoaderResponse *response);
```

## Parameters

| Name       | Type                                                   | Description    |
| ---------- | ------------------------------------------------------ | -------------- |
| `response` | [`AnimaXLoaderResponse`](./animax-loader-response) `*` | Load response. |

</PlatformTabs.Tab>

## Completion Callback

### onComplete

Returns the load response.

**Parameters**

| Name       | Type                                                    | Description    |
| ---------- | ------------------------------------------------------- | -------------- |
