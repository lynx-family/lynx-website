# AnimaXSettingService

`AnimaXSettingService` maps to Android `IAnimaXSettingService` and lets integrators provide AnimaX runtime configuration values dynamically.

## Read Configuration

### getValueByKey

Gets the configuration for a given key.

**Parameters**

| Name  | Type     | Description        |
| ----- | -------- | ------------------ |
| `key` | `String` | Configuration key. |

**Return value**

`AnimaXSettingValue`. Returns an empty value object when the key is not found or reading fails.

## Registration

```java
AnimaXServiceCenter.inst().registerService(
    IAnimaXSettingService.class,
    mySettingService);
```
