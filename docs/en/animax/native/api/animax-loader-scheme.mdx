# AnimaXLoaderScheme

`AnimaXLoaderScheme` identifies resource loading sources. Custom loaders use it to declare which type of resource they handle.

<PlatformTabs className="animax-api-platform-tabs" queryKey="platform">
<PlatformTabs.Tab platform="ios">

## Enum Values

| Enum Value                  | Value | Description        |
| --------------------------- | ----- | ------------------ |
| `AnimaXLoaderSchemeAsset`   | `0`   | App bundle asset.  |
| `AnimaXLoaderSchemeFile`    | `1`   | Local file.        |
| `AnimaXLoaderSchemeHttp`    | `2`   | HTTP resource.     |
| `AnimaXLoaderSchemeDataURL` | `3`   | Data URL resource. |

</PlatformTabs.Tab>
