## <Link href="/blog/lynx-3-5" className="sh-no-underline hover:sh-underline">Lynx 3.5: Main Thread Script, React Compiler, HarmonyOS Improvements</Link>

_November 10th, 2025_

<BlogAvatar list={['liujilong', 'huxpro', 'lynx']} />

Lynx 3.5 is now officially released! Following our stable bi-monthly release cadence, Lynx 3.5 brings several key improvements. This release adds experimental support for React Compiler, enhances main thread scripts with `stopPropagation` for better event bubbling control, enables immediate interactivity after first-frame rendering, and improves cross-thread communication. We've also expanded HarmonyOS platform support and introduced new features like the `pointer-events` CSS property.

## <Link href="/blog/lynx-3-4" className="sh-no-underline hover:sh-underline">Lynx 3.4: HarmonyOS Support, Trace and Recorder, Text Input Elements</Link>

_September 3rd, 2025_

<BlogAvatar list={['liujilong', 'lynx']} />

Lynx 3.4 is now officially released!
This release features new platform support for HarmonyOS, enables Windows development, launches new developer tools _Trace_ and _Recorder_, open-sources the highly requested `<input>` and `<textarea>` elements, offers even more flexible animations, supports variable fonts, and more.
