# Record Trace

This article provides a comprehensive guide on how to use the Devtool to record performance data for Lynx pages. By following these steps, you will be able to capture detailed information about the page rendering process and engine execution, enabling effective performance analysis and problem diagnosis.

## Setup

1. Install [Lynx Devtool Desktop Application](/guide/devtool/panels.html#run-lynx-devtool-desktop-application);
2. Use the latest [LynxExplorer app](https://github.com/lynx-family/lynx/releases) or [integrate lynx dev version](/guide/start/integrate-lynx-dev-version.html) into your application;
3. Use a data cable to connect your device to your computer;

## Record Trace

<Steps>
#### 1. Select Trace  
Open the Devtool desktop application and choose **Trace** from the topbar.

<img
  src="https://lf-lynx.tiktok-cdns.com/obj/lynx-artifacts-oss-sg/plugin/static/select-lynx-trace-panel-latest2.png"
  alt="Select Trace Panel"
  className="full_image"
/>

#### 2. Select the app

Pick the target app you want to profile from the available device list.

<img
  src="https://lf-lynx.tiktok-cdns.com/obj/lynx-artifacts-oss-sg/plugin/static/select-target-app-latest.png"
  alt="Select Target App"
  className="full_image"
/>

#### 3. Start recording

Press the **Start** to start recording trace.

<img
  src="https://lf-lynx.tiktok-cdns.com/obj/lynx-artifacts-oss-sg/plugin/static/start-trace-latest.png"
  alt="Start Trace"
  className="full_image"
/>

#### 4. Stop recording

Click the **Stop** to stop recording trace.

<img
  src="https://lf-lynx.tiktok-cdns.com/obj/lynx-artifacts-oss-sg/plugin/static/stop-trace-latest.png"
  alt="Stop Trace"
  className="full_image"
/>

#### 5. Analyze

After stopping, the trace data will be displayed for analysis.

<img
  src="https://lf-lynx.tiktok-cdns.com/obj/lynx-artifacts-oss-sg/plugin/static/analyze-trace-latest.png"
  alt="Analyze Trace"
  className="full_image"
/>

#### 6. Download trace data(optional)

If needed, click the **Download** to save the trace file to your computer.

<img
  src="https://lf-lynx.tiktok-cdns.com/obj/lynx-artifacts-oss-sg/plugin/static/download-trace-latest.png"
  alt="Download Trace"
  className="full_image"
/>

#### 7. View or manage historical trace data

Click **History** to see previous trace data.

<img
  src="https://lf-lynx.tiktok-cdns.com/obj/lynx-artifacts-oss-sg/plugin/static/history-trace-latest.png"
  alt="Trace History"
  className="full_image"
/>

You can select a trace file to **view**, **download**, or **delete** as needed on Trace History panel.

</Steps>

### FAQ

##### After clicking Start, the message "Current app does not integrate lynx and lynx-trace components with dev version, please integrate dev version first according to the following document" appears.

- Use the latest [LynxExplorer app](https://github.com/lynx-family/lynx/releases), or refer to [integrate lynx dev version](/guide/start/integrate-lynx-dev-version.html) to integrate the dev version of lynx.

##### After clicking Start, the message "Tracing already started" appears.

- Trace recording is already in progress. You can click **Stop** to stop recording trace.
