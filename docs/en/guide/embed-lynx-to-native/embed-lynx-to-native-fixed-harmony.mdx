You can give LynxView a fixed size.

For example, if you want to set a LynxView with a size of 400\*200:

```ts
// You can directly specify the size of LynxView
LynxView({}).width('400px').height('200px');
// You can also fix the parent node size and set percentage mode
LynxView({}).width('100%').height('100%');
```
