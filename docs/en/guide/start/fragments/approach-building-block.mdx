# Use as a Building Block

Use Lynx as a high-performance, cross-platform UI rendering engine within your existing application. This is the most flexible approach and supports **all Lynx platforms**. It's well-suited for enterprise applications and large-scale apps that need to progressively adopt Lynx.

### Key Advantages

- **Maximum flexibility** — Embed Lynx views anywhere in your existing app
- **All platforms** — Supports iOS, Android, HarmonyOS, and Web
- **Brownfield-friendly** — Adopt incrementally without rewriting your app
- **Enterprise-ready** — Used by large-scale applications in production
