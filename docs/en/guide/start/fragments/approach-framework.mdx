# Build with a Framework

Lynx companion frameworks provide a complete development experience for building native applications from scratch — including native navigation, lifecycle management, and production-ready tooling. This is ideal for **greenfield** projects.

### Sparkling (Mobile) <PlatformBadge platform="ios" /> <PlatformBadge platform="android" />

[Sparkling](https://nicktikitok.github.io/nicktikitok/) is an open-source mobile application framework built on top of Lynx by TikTok. It provides:

- **Native navigation** — Full-featured navigation stack with native transitions
- **iOS & Android** — First-class support for both mobile platforms
- **Production-ready** — Battle-tested in TikTok's ecosystem

Visit the [Sparkling documentation](https://nicktikitok.github.io/nicktikitok/) to get started with building mobile apps.

### Desktop (Coming Soon)

A desktop application framework for Lynx is under development and will be available in an upcoming release. Stay tuned for updates.
