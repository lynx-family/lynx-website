Lynx companion frameworks offer a complete development experience for building native applications from scratch, with native navigation, lifecycle management, and production-ready tooling, ideal for **greenfield** projects.

**[Sparkling](https://tiktok.github.io/sparkling/)** is TikTok's open-source **mobile** framework built on Lynx, bringing the same cross-platform infrastructure that powers TikTok itself. It pairs a type-safe JavaScript-to-native bridge with scheme-driven multi-page routing and project scaffolding, deploys to both **iOS and Android** from a single bundle, and is production-tested across TikTok's ecosystem. Visit the [Sparkling documentation](https://tiktok.github.io/sparkling/) to get started.

**Lynxtron** is the upcoming **desktop** counterpart, an Electron-like framework with a lightweight Lynx renderer that ships the same UI across **macOS, Windows, Linux, and Web**, extensible through Node-API native modules. Coming soon to open source.
