# Type Alias: ProtocolHandler

```ts
type ProtocolHandler = (
  request: ResourceRequest,
) => ResourceResponse | Promise<ResourceResponse> | false | undefined;
```

A function that handles a Lynx resource request for a registered URL scheme.

Return a `ResourceResponse` or `Promise<ResourceResponse>` to provide the
resource. Return `false` or `undefined` to fall back to the normal Lynx resource
loading path. The response must include `statusCode` and `Buffer` data; invalid
responses fall back to normal loading.
