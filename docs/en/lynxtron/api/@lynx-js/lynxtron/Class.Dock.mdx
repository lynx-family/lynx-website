### bounce() <span className="rp-toc-exclude api-platform-badges-inline"><Lynx.PlatformBadge platform="macos" /></span>

```ts
bounce(type?: "critical" | "informational"): number
```

an ID representing the request.

When `critical` is passed, the dock icon will bounce until either the
application becomes active or the request is canceled.

When `informational` is passed, the dock icon will bounce for one second.
However, the request remains active until either the application becomes active
or the request is canceled.

> [!NOTE] This method can only be used while the app is not focused; when the
> app is focused it will return -1.
