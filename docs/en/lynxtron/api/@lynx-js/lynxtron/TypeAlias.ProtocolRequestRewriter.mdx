# Type Alias: ProtocolRequestRewriter

```ts
type ProtocolRequestRewriter = (
  request: ProtocolRewriteRequest,
) => string | false | undefined;
```

A function that can rewrite a resource request URL before protocol handlers and
fallback loaders run.

> [!NOTE]
> This is a Lynxtron-specific API.

Return a string to use a new URL. Returning `false`, `undefined`, or any
non-string value keeps the original URL.
