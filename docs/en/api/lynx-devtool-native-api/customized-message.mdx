# CustomizedMessage

<APISummary />

Encapsulates the debugging message to be sent.

To learn how to send debugging messages, please refer to [`sendMessage`](/api/lynx-devtool-native-api/lynx-base-inspector-owner/send-message).

## Syntax

### iOS

```objective-c title="CustomizedMessage.h"
@interface CustomizedMessage : NSObject

@property(nonatomic, readwrite) NSString *type;
@property(nonatomic, readwrite) NSString *data;

@end
```

### Android

```java title="CustomizedMessage.java"
public class CustomizedMessage {
  public CustomizedMessage(String type, String data);
}
```

### Parameters

- `type`: The type of message to send, such as [CDP](https://chromedevtools.github.io/devtools-protocol/) or other custom message types.
- `data`: The message data to send.
