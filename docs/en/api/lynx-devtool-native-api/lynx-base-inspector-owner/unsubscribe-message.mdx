# unsubscribeMessage

<APISummary />

Unsubscribes from debugging messages of a specified type.

This method is the counterpart to [`subscribeMessage`](/api/lynx-devtool-native-api/lynx-base-inspector-owner/subscribe-message). You should call it to unsubscribe when you no longer need to receive a certain type of messages.

## Syntax

### iOS

```objective-c title="LynxBaseInspectorOwnerNG.h"
- (void)unsubscribeMessage:(nonnull NSString *)type;
```

### Android

```java title="LynxBaseInspectorOwnerNG.java"
void unsubscribeMessage(String type);
```

### Harmony

```ts title="LynxBaseInspectorOwner.ets"
unsubscribeMessage(type: string): void;
```

### Parameters

- `type`: The type of message to unsubscribe from.

## Usage Example

Please refer to [`subscribeMessage`](/api/lynx-devtool-native-api/lynx-base-inspector-owner/subscribe-message#usage-example).
