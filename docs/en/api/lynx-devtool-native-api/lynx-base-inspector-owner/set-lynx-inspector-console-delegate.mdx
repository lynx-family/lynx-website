# setLynxInspectorConsoleDelegate

<APISummary />

Registers a console listener for a `LynxView` instance to listen for all console messages from that instance.

## Syntax

### iOS

```objective-c title="LynxBaseInspectorOwner.h"
- (void)setLynxInspectorConsoleDelegate:(id _Nonnull)delegate;
```

### Android

```java title="LynxBaseInspectorOwner.java"
void setLynxInspectorConsoleDelegate(Object delegate);
```

### Harmony

```ts title="LynxBaseInspectorOwner.ets"
setLynxInspectorConsoleDelegate(delegate: LynxInspectorConsoleDelegate): void;
```

### Parameters

- `delegate`: An object that implements [`LynxInspectorConsoleDelegate`](/api/lynx-devtool-native-api/lynx-inspector-console-delegate#syntax).

## Usage Example

Please refer to [`LynxInspectorConsoleDelegate`](/api/lynx-devtool-native-api/lynx-inspector-console-delegate#usage-example).
