# setDevToolLibraryLoader

<APISummary />

Sets a custom library loader for DevTool, used to load the dynamic libraries (e.g., `liblynxdevtool.so`) required by DevTool at runtime.

If this method is not called, DevTool will use the default library loader.

## Syntax

### Android

```java title="LynxDevToolUtils.java"
static public void setDevToolLibraryLoader(INativeLibraryLoader loader);
```

#### Parameters

- `loader`: An instance of a custom library loader that implements the `INativeLibraryLoader` interface.
