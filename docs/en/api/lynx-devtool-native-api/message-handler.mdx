# MessageHandler

<APISummary />

A **protocol (or interface)** that acts as a handler for debugging messages.

To learn how to subscribe to debugging messages, please refer to [`subscribeMessage`](/api/lynx-devtool-native-api/lynx-base-inspector-owner/subscribe-message).

## Syntax

Implement the `MessageHandler` protocol (or interface) and receive debugging messages through the `onMessage` method.

### iOS

```objective-c title="LynxBaseInspectorOwnerNG.h"
@protocol MessageHandler <NSObject>

@required
- (void)onMessage:(nonnull NSString *)message;

@end
```

### Android

```java title="MessageHandler.java"
public interface MessageHandler {
  void onMessage(String message);
}
```

### Harmony

```ts title="MessageHandler.ets"
export interface MessageHandler {
  onMessage: (msg: string) => void;
}
```
