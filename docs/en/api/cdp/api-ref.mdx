## JavaScript Runtime Support

The runtime-domain methods in this reference are implemented by the PrimJS inspector. PrimJS is used for both Main-Thread Scripting (MTS) and Background-Thread Scripting (BTS).

V8-backed debugging is delegated to V8 Inspector. Refer to the [V8 CDP reference](https://chromedevtools.github.io/devtools-protocol/1-3) for the V8 Inspector protocol version adopted by Lynx. V8 methods are not listed individually in this reference.
