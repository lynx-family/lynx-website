# LynxValue
LynxValue is a set of C Value APIs provided by Lynx, supporting data exchange between LynxSDK and Lynx client interface users.

## lynx_value_typeof
Get the type of lynx_value.
```c
lynx_api_status lynx_value_typeof(lynx_api_env env,
                                  lynx_value value,
                                  lynx_value_type* result); 
```

## lynx_value_create_null
This API creates a lynx_value of type null.
```c
lynx_api_status lynx_value_create_null(lynx_api_env env,
                                       lynx_value* result);
```

## lynx_value_create_string_utf8
This API creates a string value from a UTF8-encoded C string. The native string is copied.This API allocates memory on the heap to store the container. After use, lynx_value_remove_reference must be called to free this portion of memory, otherwise it will result in a memory leak.
```c
lynx_api_status lynx_value_create_string_utf8(
    lynx_api_env env, const char* str, size_t length, lynx_value* result);
```

## lynx_value_create_map
This API creates a map container. This API allocates memory on the heap to store the container. After use, lynx_value_remove_reference must be called to free this portion of memory, otherwise it will result in a memory leak.
```c
lynx_api_status lynx_value_create_map(lynx_api_env env,
                                      lynx_value* result);;
```

## lynx_value_create_array

## lynx_value_get_string_utf8
Read the UTF8-encoded string value from lynx_value. lynx_api_string_expected is returned if the types do not match. If the buf argument is passed NULL, the length of the string in bytes and excluding the null terminator is returned in result
```c

                                           lynx_value value,
                                           char* buf,
                                           size_t bufsize,
                                           size_t* result);
```

## lynx_value_iterate_value
This API is used for iterating through array or map type lynx_value.
```c
lynx_api_status lynx_value_iterate_value(
    lynx_api_env env, lynx_value object, lynx_value_iterator_callback callback,
    void* func, void* raw_data);
```

## lynx_value_add_reference
Increment the reference count by 1 for pointer data.
```c
lynx_api_status lynx_value_add_reference(
    lynx_api_env env, lynx_value value, lynx_value_ref* result);
```

## lynx_value_remove_reference
Decrement the reference count by 1 for pointer data.
```c
lynx_api_status lynx_value_remove_reference(
    lynx_api_env env, lynx_value value, lynx_value_ref ref);
```
