# lynx_value_create_double

## Introduction

Create a lynx_value of type double with given value.

## Syntax

```c
lynx_api_status lynx_value_create_double(lynx_api_env env,
                                         double value,
                                         lynx_value* result);
```

### Parameters

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | The environment in which lynx value runs, typically nullptr. |
| [in] value    | Double value to be represented in lynx_value.                            |
| [out] result  | A lynx_value representing a double.                            |

### Return Value

Returns `lynx_api_ok` if the API succeeded.

## Example

```c
lynx_value val;
lynx_api_status status = lynx_value_create_double(env, 1.1f, &val);
if (status != lynx_api_ok) {
  // handle error.
}
```
