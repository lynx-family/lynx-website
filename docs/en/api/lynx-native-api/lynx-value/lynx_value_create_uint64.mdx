# lynx_value_create_uint64

## Introduction

Create a lynx_value of type uint64_t with given value.

## Syntax

```c
lynx_api_status lynx_value_create_uint64(lynx_api_env env,
                                         uint64_t value,
                                         lynx_value* result);
```

### Parameters

| Key           | Description                                                    |
|:--------------|:---------------------------------------------------------------|
| [in] env      | The environment in which lynx value runs, typically nullptr.   |
| [in] value    | Unsigned 64-bit integer value to be represented in lynx_value. |
| [out] result  | A lynx_value representing a uint64_t.                          |

### Return Value

Returns `lynx_api_ok` if the API succeeded.

## Example

```c
lynx_value val;
lynx_api_status status = lynx_value_create_uint64(env, 1000, &val);
if (status != lynx_api_ok) {
  // handle error.
}
```
