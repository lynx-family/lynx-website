# lynx_value_create_int64

## Introduction

Create a lynx_value of type int64_t with given value.

## Syntax

```c
lynx_api_status lynx_value_create_int64(lynx_api_env env,
                                        int64_t value,
                                        lynx_value* result);
```

### Parameters

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | The environment in which lynx value runs, typically nullptr. |
| [in] value    | 64-bit integer value to be represented in lynx_value.        |
| [out] result  | A lynx_value representing a int64_t.                         |

### Return Value

Returns `lynx_api_ok` if the API succeeded.

## Example

```c
lynx_value val;
lynx_api_status status = lynx_value_create_int64(env, 100, &val);
if (status != lynx_api_ok) {
  // handle error.
}
```
