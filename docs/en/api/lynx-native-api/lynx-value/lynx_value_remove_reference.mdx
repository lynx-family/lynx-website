# lynx_value_remove_reference

## Introduction

Decrement the reference count by 1 for pointer data.

## Syntax

```c
lynx_api_status lynx_value_remove_reference(
    lynx_api_env env, lynx_value value, lynx_value_ref ref);
```

### Parameters

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | The environment in which lynx value runs, typically nullptr. |
| [in] value    | lynx_value to be removed ref.                                                  |
| [out] ref     | lynx_value_ref to be remove if exist.                        |

### Return Value

Returns `lynx_api_ok` if the API succeeded.

## Example

```c
lynx_value val;
lynx_api_status status = lynx_value_create_array(env, &val);
if (status != lynx_api_ok) {
  // handle error.
}
// do something.
lynx_value_remove_reference(env, val, NULL);
```
