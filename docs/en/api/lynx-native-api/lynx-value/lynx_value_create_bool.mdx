# lynx_value_create_bool

## Introduction

Create a lynx_value of type bool with given value.

## Syntax

```c
lynx_api_status lynx_value_create_bool(lynx_api_env env,
                                       bool value,
                                       lynx_value* result);
```

### Parameters

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | The environment in which lynx value runs, typically nullptr. |
| [in] value    | Bool value to be represented in lynx_value.                  |
| [out] result  | A lynx_value representing a bool.                            |

### Return Value

Returns `lynx_api_ok` if the API succeeded.

## Example

```c
lynx_value val;
lynx_api_status status = lynx_value_create_bool(env, true, &val);
if (status != lynx_api_ok) {
  // handle error.
}
```
