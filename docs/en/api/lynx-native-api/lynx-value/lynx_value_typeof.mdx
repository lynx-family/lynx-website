# lynx_value_typeof

## Introduction

Get the type of lynx_value.

## Syntax

```c
lynx_api_status lynx_value_typeof(lynx_api_env env,
                                  lynx_value value,
                                  lynx_value_type* result); 
```

### Parameters

| Key           | Description                                                  |
|:--------------|:-------------------------------------------------------------|
| [in] env      | The environment in which lynx value runs, typically nullptr. |
| [in] value    | The lynx_value whose type to query.                          |
| [out] result  | The type of lynx_value.                                      |

### Return Value

Returns lynx_api_ok if the API succeeded.

## Example

```c
lynx_value_type type;
lynx_api_status status = lynx_value_typeof(env, value, &type);
if (status != lynx_api_ok) {
  // handle error.
}
```
