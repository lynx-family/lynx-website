# reload

<APISummary />

Reload the current page.

## Syntax

### Web

```ts
reload(): void;
```

### Desktop (C++)

```cpp
void ReloadTemplate(std::shared_ptr<lynx::pub::LynxTemplateData> data,
                    std::shared_ptr<lynx::pub::LynxTemplateData> global_props);
```

#### Parameters

- `data`: The template data used for reload.
- `global_props`: The global props used for reload. Pass `nullptr` when not needed.

### Desktop (C API)

```c
void lynx_view_reload_template(lynx_view_t* view,
                               lynx_template_data_t* data,
                               lynx_template_data_t* global_props);
```

#### Parameters

- `view`: The `lynx_view_t` instance to reload.
- `data`: The template data used for reload.
- `global_props`: The global props used for reload. Pass `NULL` when not needed.
