# updateFontScale

Changing the font scaling ratio in client settings will automatically change the text size.

## Syntax

### Android

```java
public void updateFontScale(float scale);
```

#### Parameters

- `scale`: The new font scaling ratio.

### iOS

```objc
- (void)updateFontScale:(CGFloat)scale;
```

#### Parameters

- `scale`: The new font scaling ratio.
