# updateFontScale

<APISummary />

Changing the font scaling ratio in client settings will automatically change the text size.

## Syntax

### Android

```java
public void updateFontScale(float scale);
```

#### Parameters

- `scale`: The new font scaling ratio.

### iOS

```objc
- (void)updateFontScale:(CGFloat)scale;
```

#### Parameters

- `scale`: The new font scaling ratio.

### Desktop (C++)

```cpp
void SetFontScale(float scale);
```

#### Parameters

- `scale`: The new font scaling ratio.

### Desktop (C API)

```c
void lynx_view_set_font_scale(lynx_view_t* view, const float& font_scale);
```

#### Parameters

- `view`: The `lynx_view_t` instance to update.
- `font_scale`: The new font scaling ratio.
