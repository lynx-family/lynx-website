# onFlushFinish

Called when UI flush end in async render mode.

## Syntax

### Android

```java
public void onFlushFinish(FlushInfo flushInfo);
```

#### Parameters

- `flushInfo`: Contains the begin and end timings of the flush, and whether it is a synchronous flush.
