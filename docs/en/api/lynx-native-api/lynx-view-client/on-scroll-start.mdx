# onScrollStart

Called when the UI start scrolling.

## Syntax

### Android

```java
public void onScrollStart(ScrollInfo info);
```

#### Parameters

- `info`: Contains scroll parameters, including the instance of the scrolling view, and the front-end specified id and name.
