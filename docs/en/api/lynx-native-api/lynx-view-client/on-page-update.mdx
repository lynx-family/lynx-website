# onPageUpdate

Called when page update.

## Syntax

### Android

```java
public void onPageUpdate();
```

### iOS

```objc
- (void)lynxViewDidPageUpdate:(LynxView *)view;
```

#### Parameters

- `view`: The LynxView instance.

### Harmony

```typescript
public onPageUpdate(): void;
```
