# onRuntimeReady

Called when JS environment preparation completed.

## Syntax

### Android

```java
public void onRuntimeReady();
```

:::note

- The callback is in an asynchronous thread.

:::

### iOS

```objc
- (void)lynxViewDidConstructJSRuntime:(LynxView *)view;
```

#### Parameters

- `view`: The LynxView instance.
