# onRuntimeReady

<APISummary />

Called when JS environment preparation completed.

## Syntax

### Android

```java
public void onRuntimeReady();
```

:::note

- The callback is in an asynchronous thread.

:::

### iOS

```objc
- (void)lynxViewDidConstructJSRuntime:(LynxView *)view;
```

#### Parameters

- `view`: The LynxView instance.

### Harmony

```typescript
public onRuntimeReady(): void;
```

### Desktop (C++)

```cpp
class YourViewClient : public lynx::pub::LynxViewClient {
 public:
  void OnRuntimeReady() override;
};
```

### Desktop (C API)

```c
typedef void (*on_runtime_ready)(lynx_view_client_t* client);
void lynx_view_client_bind_on_runtime_ready(lynx_view_client_t* client,
                                            on_runtime_ready callback);
```
