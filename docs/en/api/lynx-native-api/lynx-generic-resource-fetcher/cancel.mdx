# cancel

<APISummary />

Cancel the request of the requiring resource.

## Syntax

### Android

```java
public void cancel(LynxResourceRequest request);
```

#### Parameters

- `request`: the requiring request.

:::note

- This method is optional to be implemented.

:::

### Harmony

```typescript
abstract cancel(request: LynxResourceRequest): void;
```

#### Parameters

- `request`: Request to be canceled.

### Desktop (C++)

Desktop public API cancels a resource fetch by request ID:

```cpp
virtual void Cancel(lynx_resource_request_id request_id);
```

#### Parameters

- `request_id`: The ID of the request to cancel.

### Desktop (C API)

```c
typedef void (*cancel_fetch_func)(lynx_generic_resource_fetcher_t* fetcher,
                                  lynx_resource_request_id request_id);
void lynx_generic_resource_fetcher_bind_cancel_fetch(
    lynx_generic_resource_fetcher_t* fetcher,
    cancel_fetch_func callback);
```

#### Parameters

- `fetcher`: The `lynx_generic_resource_fetcher_t` instance.
- `request_id`: The ID of the request to cancel.
