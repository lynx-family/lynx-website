## Introduction

The `-x-caret-width` property sets the width of the text insertion caret in editable elements.

## Syntax

```css
input {
  -x-caret-width: 2px;
}
```

### Values

- [`<length>`](/api/css/data-type/length.mdx)<br/>
  Sets the width of the caret.
