## Introduction

The `-x-caret-gradient` property sets the gradient used to paint the text insertion caret in editable elements.

## Syntax

```css
input {
  -x-caret-gradient: linear-gradient(to bottom, #3b82f6, #a855f7);
}
```

### Values

- `none`<br/>
  Uses the default caret paint.

- [`<gradient>`](/api/css/data-type/gradient.mdx)<br/>
  Sets the gradient brush for the caret.
