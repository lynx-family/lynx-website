# lynx: performance readonly object

This object provides access to performance metrics related to the current page.

## Instance Methods

[`createObserver()`](./lynx-performance/create-observer) Creates a [`PerformanceObserver`](/api/lynx-api/performance-api/performance-observer) object that will be used to receive performance events generated by Lynx engine.

[`profileStart()`](./lynx-performance/profile-start) Marks the start of a [trace event](https://perfetto.dev/docs/instrumentation/track-events).

[`profileEnd()`](./lynx-performance/profile-end) Marks the end of the most recent [trace event](https://perfetto.dev/docs/instrumentation/track-events).

[`profileMark()`](./lynx-performance/profile-mark) Marks an instant(point-in-time) event.

[`profileFlowId()`](./lynx-performance/profile-flow-id) Generates a unique flow id for [flow events](https://perfetto.dev/docs/instrumentation/track-events#flows).

[`isProfileRecording()`](./lynx-performance/is-profile-recording) Checks if trace recording is currently active.
