# console: debug() method

`debug` outputs a message to the console at the "debug" log level.

## Syntax

```ts
debug(val1, val2, ...): undefined;
```

### Parameters

#### val1, val2, ...

A list of JavaScript values to output.

### return value

None(`undefined`).
