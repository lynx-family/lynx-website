# console: info() method

`info` 方法用于在控制台面板输出信息，日志级别为 info 级别。

## Syntax

```ts
info(val1, val2, ...): undefined;
```

### Parameters

#### val1, val2, ...

输出的JavaScript值列表。这些值中的每个值的表示形式都以给定的顺序输出。

### return value

None(`undefined`).
